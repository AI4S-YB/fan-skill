# 转座子分析结果解读

## 一句话解释

转座子分析：鉴定和注释基因组中的可移动元件，研究其类型、分布和进化历史。

## 能回答什么

- 基因组中有多少TE？
- 主要的TE类型是什么？
- TE的染色体分布如何？
- 什么时候发生了TE扩增？
- TE与基因有什么关系？

## 不能回答什么

- TE的具体功能？（需要实验验证）
- TE的插入目标位点？（需要实验或精细分析）
- TE的移动机制细节？（需要分子生物学研究）

## 典型输出

| 文件 | 含义 |
|------|------|
| `TElib.fa` | TE序列库 |
| `TEanno.gff` | TE注释文件 |
| `TEstats.txt` | TE统计信息 |
| `insertion_times.pdf` | 插入时间分布图 |
| `TElandscape.pdf` | TE景观图 |
| `distribution.pdf` | 染色体分布图 |

## 常见结果模式

### 高LTR比例

植物基因组通常LTR逆转录转座子占主导，尤其是Copia和Gypsy超家族。Gypsy倾向于着丝粒区域，Copia分布更均匀。

### 近期TE爆发

插入时间分布显示近期有大量TE插入，提示TE爆发事件。这常见于植物适应性进化过程中。

### 基因邻近TE富集

某些TE类型（如MITE）在基因附近富集，可能影响基因表达调控。

### TE景观图特征

景观图横轴为分化距离（时间代理），纵轴为覆盖度。峰代表TE扩增事件。

## 植物特异性结果解读

### 基因组大小与TE

植物基因组大小主要由TE决定。高TE比例导致大基因组（如小麦17Gb），低TE比例导致小基因组（如拟南芥135Mb）。

### LTR类型分布

Gypsy在植物着丝粒区域富集，可能与异染色质形成相关。Copia在基因间区更常见。

### 近期扩增

植物常有近期TE扩增，与环境适应、驯化相关。注意区分真实扩增和组装质量影响。

2. 展示TE景观图
3. 分析主要TE超家族的分布
4. 如有插入时间分析，展示时间分布
5. 讨论TE与基因组的关联
6. 与近缘物种比较TE组成
