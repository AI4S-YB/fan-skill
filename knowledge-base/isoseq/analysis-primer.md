# Iso-Seq全长转录组分析结果解读

## 一句话解释

Iso-Seq分析：获得全长转录本序列，准确鉴定转录本结构和可变剪接事件。

## 能回答什么

- 有多少全长转录本？
- 转录本结构是什么（外显子-内含子边界）？
- 有多少可变剪接事件？
- 是否有新的转录本？
- 是否有融合基因？

## 不能回答什么

- 转录本的精确表达量？（需要Illumina数据补充）
- 组织特异性？（需要多样本实验设计）
- 功能验证？（需要实验研究）

## 典型输出

| 文件 | 含义 |
|------|------|
| `hq_transcripts.fasta` | 高质量全长转录本 |
| `classification.txt` | SQANTI3分类结果 |
| `junctions.txt` | 剪接位点信息 |
| `splicing_events.ioe` | 可变剪接事件 |
| `fusion_candidates.txt` | 融合基因候选 |

## 常见结果模式

### 高FSM比例

大部分转录本匹配已知注释，说明Iso-Seq数据质量好，注释数据库完善。

### 大量NIC/NNC转录本

发现新转录本，可能是物种特异或条件特异的转录本。需要验证。

### 内含子保留普遍

内含子保留是植物最常见的可变剪接类型，比例可能高于动物。

### PolyA长度分布

PolyA尾长度分布可以反映mRNA稳定性信息。

## 植物特异性结果解读

### 剪接模式差异

植物内含子保留比例较高，而动物外显子跳跃更常见。

### 多倍体同源转录本

多倍体物种可能有多套相似的转录本，需要区分同源基因。

### 长非编码RNA

Iso-Seq适合发现植物长链非编码RNA，这些在植物研究中较少。

## 结果报告建议

1. 报告CCS数量和全长比例
2. 统计HQ/LQ转录本数量
3. 展示SQANTI3分类结果
4. 报告发现的剪接事件类型和数量
5. 如有新转录本，进行功能注释
6. 讨论发现的生物学意义
