# 植物比较基因组学：分析笔记

## 在开始之前

比较基因组学回答进化问题：这个基因从哪里来？基因组什么时候加倍过？哪些基因在快速进化？

### 你分析的是什么尺度的进化？

- **种内**（同一物种的不同个体/品种）→ MCScanX 自比对，识别品种间结构变异
- **近缘种**（如水稻 vs 高粱，~50 Mya 分化）→ MCScanX 双向比对，Ks ≈ 0.5-1.0
- **远缘种**（如水稻 vs 拟南芥，~150 Mya）→ OrthoFinder 更合适，共线性可能不显著

---

## 植物全基因组加倍 (WGD) 历史

几乎所有植物都经历过多次 WGD：
- **γ (gamma)**: ~170 Mya，所有被子植物共有
- **β (beta)**: ~130 Mya，核心真双子叶植物
- **ρ (rho)**: ~70 Mya，禾本科特有（水稻、玉米、小麦、高粱共同经历）
- **α (alpha)**: ~50 Mya，十字花科（拟南芥、油菜）
- **大豆**: ~13 Mya 和 ~58 Mya 两次 WGD

→ 如果某个物种的基因家族异常大，先查是不是 WGD 造成的。

### WGD 历史对分析的实际影响

**γ (gamma) 三倍化**是所有被子植物基因组分析的基线。这个事件的痕迹在所有被子植物基因组中都能看到（尽管在约 170 Mya 的时间尺度上信号已经模糊）。如果你在分析一个被子植物基因组，任何"大基因家族"都至少部分源于 γ。

具体到主要作物：
- **水稻**：ρ (禾本科) 之后无额外的近期 WGD。水稻基因组是"进化上安静的"——基因家族相对紧凑。ρ 事件产生的同源区域仍然清晰可辨。
- **玉米**：ρ 之后再无 WGD，但玉米在 ~5-12 Mya 发生了异源四倍体事件（玉米是古四倍体，已经二倍化）。玉米基因组中仍然可以看到两个"亚基因组"的痕迹（maize1 和 maize2）。
- **小麦**：禾本科 ρ 之后，小麦谱系经历了两次独立的 WGD（第一次产生 AA/BB 分化，第二次加入 DD 基因组）——形成异源六倍体 AABBDD。三个亚基因组之间的基因保留和丢失 (gene fractionation) 模式是小麦基因组学研究的核心课题。
- **大豆**：在双子叶植物 β 之后，大豆经历了两次近期 WGD (~13 Mya 和 ~58 Mya)。~13 Mya 的那次 WGD 产生的同源区域大约有 75% 的基因保留，这意味着大豆中约 75% 的基因是多拷贝的。
- **棉花**：At 和 Dt 亚基因组在 ~1-2 Mya 合并形成异源四倍体。这是极其近期的事件——两个亚基因组之间的差异非常小（类似同一物种的不同生态型），但它们的表达调控和表观遗传修饰已经有显著的分化。
- **油菜**：与棉花类似，约 7500 年前 A 和 C 亚基因组合并。油菜的"天然"和"合成"异源四倍体比较是研究多倍体早期进化的绝佳系统。

**实际分析建议**：
- 在注释基因家族时，始终要有一个 WGD 时间线作为背景。一个植物有 100 个拷贝的热激蛋白基因并不奇怪——它可能是 1（祖先）× 2（γ）× 2（β）× 2（ρ）× 2（近期 WGD）+ 若干串联重复的结果。
- 要注意 WGD 后基因的不对称丢失（biased fractionation / gene loss）。在多倍体中，通常来自一个亚基因组的基因比来自另一个亚基因组的基因更容易丢失。这导致多倍体基因组中不同亚基因组的"基因密度"不同。

## 共线性分析的关键选择

- **种内共线性** → 能直接看到 WGD 的痕迹（如拟南芥的 4 个同源区域跨越 5 条染色体）
- **种间共线性** → 揭示染色体进化的保守性（禾本科的染色体结构在 70 Mya 内高度保守）
- **多倍体** → 亚基因组之间的共线性更强；亚基因组与其他二倍体近缘种之间的共线性退化更慢

### 多倍体共线性分析：亚基因组感知 vs 合并方法

在分析多倍体（如小麦、棉花、油菜）的共线性时，有两个根本不同的策略：

**亚基因组感知方法（推荐）**：
1. 将多倍体基因组拆分为各个亚基因组（如小麦 → TaA, TaB, TaD）
2. 对每个亚基因组分别和近缘二倍体物种做共线性分析（TaA vs T. urartu, TaB vs Aegilops speltoides, TaD vs Aegilops tauschii）
3. 再对亚基因组之间做共线性分析（TaA vs TaB vs TaD）
4. 最后将亚基因组内和亚基因组间的共线性块整合

**优点**：能将 WGD 产生的共线性（古老的，所有亚基因组共享）与多倍体化后亚基因组间的共线性（较新的）区分开来。这是"教科书级"的分析策略。

**缺点**：需要高质量的亚基因组分离注释。

**合并方法（简化，不得已时使用）**：
- 将整个多倍体基因组作为一个"合并二倍体"进行共线性分析。
- 缺点：亚基因组间的同源关系（homeology）会被错误地解释为种内的旁系同源（paralogy）。你无法区分"基因组内由于 WGD 产生的共线性"和"亚基因组间由于多倍体化产生的共线性"。

**实际建议**：如果用户是做小麦/棉花/油菜的共线性分析，且注释中已区分亚基因组来源（如 TraesCS1A... vs TraesCS1B... vs TraesCS1D...）→ 用亚基因组感知方法。如果用户无法获得亚基因组特异注释 → 诚实说明合并方法的局限性。

### 共线性分析工具选择

| 工具 | 适合 | 不适合 |
|------|------|--------|
| MCScanX | 二倍体种内/种间共线性。植物社区的标准工具。 | 多倍体需要手动拆分亚基因组 |
| JCVI (Python) | 与 MCScanX 类似的算法，更好的 Python 生态集成 | 速度略慢于 MCScanX |
| SynMap (CoGe) | 在线，可视化出色。适合探索性分析 | 大规模分析（>全基因组）可能较慢 |
| i-ADHoRe | 检测古老/退化共线性区域 | 需要参数调优 |
| GENESPACE | 跨物种的"泛基因组共线性"分析 | 需要较多物种的注释 |

## Ks 分析 — 基因组加倍的"分子时钟"

Ks（每个同义位点的同义替换数）随时间大致线性累积。

### 怎么读 Ks 峰值

- **尖锐的单峰在 Ks ≈ 0.1-0.3** → 最近的一次 WGD（或物种形成）
- **宽的峰在 Ks ≈ 0.8-1.5** → 较古的 WGD
- **多个峰** → 多次 WGD 事件叠加
- **Ks > 2.0 且拖尾很宽** → 已达饱和，无法精确定年

### Ks 峰值的深度解读：WGD vs 小规模重复

这是一个经典陷阱。Ks 分布中的一个峰不一定代表 WGD。

**WGD 的 Ks 信号**：
- 所有染色体上都有共线性的 duplicated blocks 贡献了这个峰。
- 峰比较尖锐（但 WGD 事件本身可能有持续时间）。
- 峰的位置在所有染色体之间一致。

**小规模串联重复的 Ks 信号**：
- 由局部的、非共线性的 duplicated gene pairs 贡献。
- 通常没有一个"全局"的峰，而是在不同染色体上有不同的模式。
- 如果只有小规模重复 → Ks 分布通常呈 L 形（近期的重复多 + 长期的指数衰减），而非一个清晰的峰。

**区分策略**：
1. 分别提取共线性 block 内和共线性 block 外的 duplicated gene pairs。
2. 对于共线性 block 内的 pairs 做 Ks 分布 → 这个分布应该反映 WGD 历史。
3. 对于共线性 block 外的 pairs 做 Ks 分布 → 这个分布反映的是串联重复 + 逆转录转座 + 其他局部机制。
4. 只有共线性 block 内的 Ks 集中分布 (peak) 才能作为 WGD 的证据。

**报告的严谨措辞**：
- "共线性同源基因对的 Ks 分布在 0.8-1.0 之间呈集中的单峰，支持一个约 XXX Mya 的 WGD 事件。"
- "非共线性同源基因对的 Ks 分布呈 L 形，表明持续但非剧增的小规模基因重复。"
- 如果只分析了总的 duplicate gene pairs（没有区分共线性/非共线性）→ "Ks 分布的峰值提示可能存在 WGD 事件，但需要共线性分析确认。"

### 常见陷阱

- 不要只看一个峰就说有 WGD → Ks 峰也可能来自小规模的片段重复
- 近交物种的 Ks 估计可能有偏 → 用 YN 模型而非 NG 模型
- 不同物种的 Ks 不能直接比较（不同谱系有不同的同义替换率）

## 直系同源推断：OrthoFinder 在植物中的特殊考虑

OrthoFinder 是植物比较基因组学中最常用的直系同源推断工具。但植物的全基因组加倍历史给直系同源推断带来了独特的挑战。

### 植物中的"旁系同源爆炸"

WGD 产生了大量的旁系同源基因 (paralogs)。在人类/哺乳动物中，大多数基因是单拷贝的——你的直系同源推断很大程度上是一个二分图匹配问题。在植物中（特别是大豆、小麦、玉米），一个"基因"在物种内可能有 2-8 个拷贝（paralogs），而每个拷贝在另一个物种中可能又有 1-4 个直系同源 (orthologs)。

**后果**：
- OrthoFinder 产生的"直系同源群 (orthogroup)"通常非常大——包含来自各物种的数十到数百个基因。
- 这些"大的 orthogroup"不是错误——它们正确反映了 WGD + 串联重复的历史。
- 但是：orthogroup 内的成员关系复杂——不是所有成员之间都是 1:1 直系同源。一个来自拟南芥的基因和一个来自大豆的基因可能在同一个 orthogroup 中，但它们是旁系同源关系而非直系同源。

### OrthoFinder 结果的解读策略

**Orthogroup 层面分析**：
- 适合回答："这个基因家族在物种之间是如何分布的？"
- 每个 orthogroup 的大小 = 各物种中该家族的基因拷贝数
- 如果一个 orthogroup 在某个物种中异常大 → 该物种中发生了该家族的扩张

**基因层面分析（需要进一步提取）**：
- 从 OrthoFinder 输出的 `Orthologues/` 目录中提取物种间的 1:1 直系同源关系
- 1:1 orthologs 是共线性分析、dN/dS 分析、功能推断的基础
- 对于 WGD 丰富的物种，1:1 ortholog 很少 → 你需要接受"1:many"或"many:many"的直系同源关系

**多倍体的推荐处理**：
- 对多倍体物种：先按亚基因组拆分蛋白质组（如 TaA.fa, TaB.fa, TaD.fa），再分别与二倍体近缘种一起跑 OrthoFinder。
- 或者：在 OrthoFinder 结果中，基于共线性和序列相似性确定亚基因组归属。

### 参考物种选择策略

不同分析问题的"最佳"参考物种不同：

| 分析目标 | 推荐参考物种 | 原因 |
|---------|------------|------|
| 单子叶功能推断 | 水稻 (Oryza sativa) | 最好的单子叶功能基因组学资源 |
| 双子叶功能推断 | 拟南芥 (Arabidopsis thaliana) | 最好的双子叶功能基因组学资源 |
| 豆科进化 | 大豆 (Glycine max) + 蒺藜苜蓿 (Medicago truncatula) | 大豆有近期 WGD，蒺藜苜蓿没有——对比可解耦 WGD 效应 |
| 禾本科共线性 | 水稻 + 高粱 + 短柄草 | 三者均为非多倍体，有完整的禾本科共线性图 |
| 十字花科进化 | 拟南芥 + 琴叶拟南芥 (A. lyrata) + 亚麻荠 (Camelina) | 不同时间尺度的多倍化事件 |
| 茄科进化 | 番茄 + 马铃薯 + 辣椒 | 二倍体 vs 同源多倍体比较 |
| 泛基因组 | 物种自身的多个 de novo 组装 | 种内结构变异和 PAV |

**不要做的事**：
- 不要用拟南芥作为水稻/玉米/小麦的"最近"参考物种——它们在约 150 Mya 分化。功能和共线性在这么远的距离上只能提供最粗略的指导。
- 不要假设"拟南芥有这个基因 → 我的物种也有"——γ WGD 后各谱系的基因保留/丢失模式不同，拟南芥有某些基因不代表你的物种有。
- 如果要推断功能：序列相似性是一个必要的条件，但不是充分的。同一个基因家族的不同成员可以执行完全不同的功能。

## 基因家族扩张

植物最大的基因家族通常与以下相关：
- **抗病** (NBS-LRR)：数百到上千个基因
- **转录因子** (MYB, bHLH, WRKY, NAC)：调控发育和胁迫响应
- **次生代谢** (CYP450, UGT)：合成防御化合物

基因家族扩张 ≠ 功能重要性。有些大基因家族是"进化的垃圾"（串联重复的假基因化产物）。

### 基因家族扩张的深入解读：扩张 ≠ 功能重要

这是一个需要纠正的常见误解。

**扩张是真实的生物学信号吗？**
- WGD 产生的扩张 (genome-wide, systemic) → 不一定是功能驱动。许多 WGD 产生的拷贝在百万年内被假基因化而后丢失。留下来的拷贝可能是因为剂量平衡约束 (gene balance hypothesis) 或新功能化 (neofunctionalization)。
- 串联重复产生的扩张 (local, tandem) → 更可能是功能驱动的。串联重复通常发生在"热点"区域，某些基因家族（如 NBS-LRR, CYP450）在这些区域积累了数十到数百个拷贝。这种扩张几乎总是和适应性进化相关。

**如何判断扩张是否有功能意义**：
1. 检查基因家族内的 dN/dS 比率。如果多数拷贝的 dN/dS > 1 → 正选择驱动扩张。如果多数拷贝 dN/dS < 0.5 → 可能是中性或纯化选择。
2. 检查表达模式。如果各拷贝在不同组织/条件下表达（表达分化）→ 亚功能化，功能重要。如果多数拷贝不表达 → 可能是假基因化产物。
3. 检查基因组位置。串联重复簇 (tandem array) → 更可能是功能驱动。散布的、全基因组分布的拷贝 → 更可能是 WGD 的遗留。
4. 检查是否存在一个"主要拷贝"（表达量最高、dN/dS 最低）+ 若干"次要拷贝"→ 典型的部分功能冗余模式。

**报告建议**：
- 不要只报告"基因家族 X 在物种 Y 中有 50 个拷贝，比拟南芥的 10 个多，因此 Y 物种的该功能得到了加强。"
- 应该报告："基因家族 X 在物种 Y 中有 50 个拷贝（vs 拟南芥 10 个），其中 30 个位于 3 个串联重复簇中，15 个在叶片中不表达（可能假基因化），5 个在花中特异性表达。dN/dS 分析显示串联重复簇中的拷贝经历了正选择 (dN/dS > 1)，表明该扩张可能具有适应性意义。"

## 选择压力分析

- **dN/dS > 1** → 正选择（适应性进化）。防御基因、生殖相关基因最常见。
- **dN/dS ≈ 1** → 中性进化。假基因或放松选择的重复基因。
- **dN/dS < 0.1** → 强纯化选择。持家基因、光合作用基因。

### 植物特有的模式

- NBS-LRR 基因：dN/dS 往往在 LRR 结构域 > 1（病原识别位点在快速进化）
- 自交不亲和位点 (S-locus)：dN/dS 极高
- 多倍化后的重复基因：dN/dS 逐步变化 — 一开始放松（≈1），然后一个拷贝恢复纯化选择（<0.1），另一个保持放松或正选择

### dN/dS 深度解读：不同基因家族的典型值

理解特定基因家族在植物中的"正常" dN/dS 范围有助于判断某个分析结果是否需要特殊解释。

| 基因家族/类别 | 典型 dN/dS 范围 | 原因 |
|-------------|--------------|------|
| 光合作用 (Rubisco, LHC) | 0.05-0.15 | 极强的纯化选择——光系统稍有变化即致死 |
| 核糖体蛋白 | 0.02-0.1 | 最强的纯化选择之一 |
| 组蛋白 | 0.05-0.2 | 极强纯化选择 |
| 持家酶 (GAPDH, Actin) | 0.05-0.2 | 强纯化选择 |
| 一般转录因子 (不涉及胁迫) | 0.1-0.3 | 中等纯化选择 |
| 胁迫响应转录因子 (ERF, WRKY) | 0.2-0.5 | 稍有放松 |
| CYP450 (一般) | 0.2-0.6 | 次生代谢酶的纯化选择较弱 |
| CYP450 (某些亚家族) | 0.5-1.2 | 可能有正选择 |
| NBS-LRR (整体基因) | 0.3-0.7 | 整体上是纯化选择 |
| NBS-LRR (LRR 结构域) | 0.8-2.5 | LRR 结构域是病原识别的界面——强正选择 |
| 花粉识别/自交不亲和 | 1.0-4.0 | 极强的正选择 |
| 已知假基因 | 0.8-1.2 | 接近中性预期 |
| WGD 重复基因（近期，<5 Mya） | 0.3-0.8 | 放松选择 |
| WGD 重复基因（古老，>50 Mya） | 0.1-0.3 | 其中一个拷贝恢复了纯化选择 |

**dN/dS 分析的陷阱**：
- **饱和度**：如果 Ks > 2（同义位点已饱和），dN/dS 估计不可靠。植物 WGD 研究中常见——γ 事件 (~170 Mya) 的 Ks > 2，此时任何基于 dN/dS 的结论都需要极其谨慎。
- **全基因 vs 结构域**：全基因 dN/dS 是一个平均值——某个结构域可能经历正选择 (dN/dS > 1)，而基因其余部分经历纯化选择 (dN/dS < 0.3)。仅报告全基因 dN/dS 会掩盖真实的选择信号。始终做滑动窗口 dN/dS 或结构域级别分析（特别是对 NBS-LRR 基因）。
- **分枝模型 vs 位点模型**：PAML 的 branch model 测试谱系特异的正选择（"某个物种的该基因是否经历了加速进化"）。site model 测试位点特异的正选择（"基因的某些位点是否在所有物种中都经历正选择"）。它们是互补的，不是替代的。
- **基因转换**：串联重复基因簇中的基因转换会"同步化"序列——dN/dS 会被错误地压低。如果你的分析中有 tandem array 的基因——检查是否发生了基因转换。

## 用拟南芥/水稻注释你的物种

**拟南芥 (Arabidopsis thaliana)** 是植物功能基因组学的"金标准参考"：
- 优点：最全面的突变体资源、表达图谱 (eFP Browser)、蛋白互作网络。
- 缺点：拟南芥是一种短命杂草——很多拟南芥的功能注释不适用于作物（如产量、籽粒品质、多年生特性相关的基因在拟南芥中没有直接的对应）。
- 十字花科内（拟南芥、油菜、白菜、甘蓝）：推断可靠。
- 豆科内（大豆、苜蓿）：拟南芥同源基因推断有一定参考价值但需谨慎。
- 禾本科内（水稻、玉米、小麦）：拟南芥同源推断的可靠性低。优先使用水稻作为参考。

**水稻 (Oryza sativa)** 是禾本科的"拟南芥"：
- 优点：高质量的突变体资源、T-DNA 插入库、全生育期表达图谱。
- 对禾本科的推断可靠性远高于拟南芥。
- 对不同亚种的适用性：japonica (Nipponbare) 的注释最完整，但 indica 可能有不同的等位基因和基因存在/缺失变异 (PAV)。

**其他有用的参考**：
- **玉米**：C4 光合作用、杂种优势、转座子生物学
- **番茄**：果实发育、成熟调控
- **大豆**：共生固氮、油脂代谢
- **杨树**：木材形成、多年生性
- **小立碗藓 (Physcomitrium patens)**：植物陆生化、基因靶向效率高
