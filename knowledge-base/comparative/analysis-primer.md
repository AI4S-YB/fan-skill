# 比较基因组学结果解读

## 一句话解释

比较基因组学通过比对不同物种/品种的基因组，揭示进化过程中基因组的保守性和变化。

## 能回答什么

- 这个基因组经历过几次全基因组加倍（WGD）？
- 某个基因家族在进化中是扩张了还是缩小了？
- 哪些基因受到正选择？
- 不同物种间染色体的共线性如何？

## 不能回答什么

- Ks 峰精确对应什么年代？（只能给出大致范围，Ks在不同谱系中的速率不同）
- 基因家族扩张一定是功能重要的吗？（可能是中性进化）
- 正选择基因一定与适应性相关吗？（需要功能验证）

## 典型输出

| 文件 | 含义 |
|------|------|
| `synteny_blocks.tsv` | 共线性区块坐标 |
| `ks_distribution.pdf` | Ks 分布图：峰值提示 WGD 事件 |
| `orthogroups.tsv` | 直系同源群分类 |
| `dn_ds_results.csv` | 各基因/基因对的选择压力 |

## 常见结果模式

### 多个 Ks 峰 → 多次 WGD
几乎所有被子植物都有 γ (170 Mya) 事件。禾本科额外有 ρ，十字花科额外有 α。

### 基因家族巨大但不一定功能重要
WGD 产生的扩张 ≠ 功能驱动。串联重复扩张更有功能意义。检查表达和 dN/dS。
