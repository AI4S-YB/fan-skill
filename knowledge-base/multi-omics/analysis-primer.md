# 多组学整合结果解读

## 一句话解释

多组学整合：同时分析多种分子数据（mRNA、蛋白、代谢物等），发现跨层次的生物学协调变化。

## 能回答什么

- 哪些生物学过程在多个组学层面上协调变化（MOFA 因子）？
- 哪些基因/蛋白/代谢物最能区分实验分组（DIABLO）？
- 两组学之间有什么关联结构（sPLS）？

## 不能回答什么

- 组学间的关联是因果关系吗？（MOFA 因子是统计结构，非因果推断）
- mRNA 和蛋白的弱相关意味着什么？（植物中 r=0.4-0.6 是正常的，翻译后调控重要）
- 未整合的组学层面有重要信息吗？（整合可能丢失组学特异信号）

## 典型输出

| 文件 | 含义 |
|------|------|
| `mofa_factors.csv` | MOFA2 因子得分和方差分解 |
| `diablo_classification.pdf` | DIABLO 样本判别图 |
| `omics_correlation_network.pdf` | 跨组学关联网络 |

## 常见结果模式

### mRNA-蛋白相关性弱 (r < 0.4)
植物中正常。翻译后调控和蛋白降解是重要因素。报告 r 值但不声称"异常"。

### MOFA 因子数难以确定
当 N<20 时限制因子数 ≤5。因子解释方差应 >5%。
