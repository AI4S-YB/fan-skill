# 群体遗传结构结果解读

## 一句话解释

群体遗传结构分析：用全基因组标记揭示样本间的遗传关系，识别亚群和遗传背景。

## 能回答什么

- 样本分为几个遗传亚群？
- 哪些样本遗传上更接近？
- 群体结构是否会影响 GWAS（λ 是否偏高）？

## 不能回答什么

- 为什么会有这些亚群？（需结合地理、历史、育种信息）
- 不同亚群在表型上有差异吗？（需结合表型数据）
- 最佳的 K 值是什么？（ADMIXTURE 的 K 选择有不确定性）

## 典型输出

| 文件 | 含义 |
|------|------|
| `pca_plot.pdf` | PCA 图：前几个主成分的样本分布 |
| `admixture_plot.pdf` | ADMIXTURE 祖先成分分配图 |
| `kinship_heatmap.pdf` | 亲缘关系热图 |
| `phylogenetic_tree.pdf` | NJ/ML 系统发育树 |

## 常见结果模式

### PC1 解释方差 > 30%
极强的群体分化。可能存在亚种级别分离（如水稻 indica×japonica）。GWAS 前必须分亚群或加 K 矩阵。

### PC1 解释方差 < 5%
群体均匀，结构弱。对 GWAS 很有利。对现代育种材料正常。

### K=1 时 CV error 最低
群体无亚群结构。正常。不强制必须有亚群。
