# 植物群体遗传学：分析笔记

## 在开始之前

植物群体遗传学分析要回答的核心问题：
1. 这些样本之间有没有遗传亚群？
2. 有几个亚群？
3. 亚群之间差异多大？
4. 哪些样本属于哪个亚群？

---

## LD Pruning 为什么重要

PCA 和 ADMIXTURE 假设 SNP 是独立的。LD 会导致某些基因组区域的信号被过度放大。

- 高密度标记（>50K SNP）：必须 pruning，用默认参数（window=50, step=5, r2=0.2）
- 低密度标记（<10K SNP）：r² 阈值适当放宽（0.3-0.5），否则 prune 掉太多标记
- 如果 LD pruning 移除了所有 SNP（可能 < 100 剩余）：降低 r² 到 0.5 或跳过 pruning

## PCA 结果怎么看

### PC1 解释方差

- **> 30%**：极强的群体分化。可能存在亚种级别或不同育种群的分离。检查样本元数据（品种类型、地理来源）。
- **15-30%**：明显的群体结构。通常对应不同育种项目、生态型或地理区域。
- **5-15%**：适度结构。可能与育种历史或微弱的地理隔离有关。
- **< 5%**：群体基本均匀（对自交作物很少见）。

### 植物的"正常"范围

- 野生种质资源的 PC1 通常 > 20%（地理隔离强）
- 现代育种材料的 PC1 通常 5-15%（种质交流频繁）
- 如果 PC1 > 30%，优先分亚群再做下游分析

### PCA 深度解读：PC1% 在不同物种中意味着什么

PC1 的解释方差百分比本身就是一个有价值的生物学信息。不同作物因为不同的进化历史和育种历史，"正常"的 PC1% 差异很大。

**按物种的典型 PC1 解释方差：**

| 物种/群体 | 典型 PC1% | 主要分化因素 |
|-----------|----------|------------|
| 水稻 (indica × japonica 混合) | 30-50% | 亚种分化 (~0.5 Mya) |
| 水稻 (纯 indica) | 10-20% | 地理起源 + 育种群 |
| 玉米 (温带 × 热带) | 15-25% | 适应区 + 育种群 |
| 玉米 (纯温带) | 5-10% | 杂种优势群 (SS vs NSS) |
| 小麦 (全球现代品种) | 8-15% | 春性/冬性 + 地理区域 |
| 小麦 (地方品种 + 现代品种) | 20-35% | 地方品种 vs 现代品种 + 地理 |
| 大豆 (北美) | 5-10% | MG (maturity group) |
| 大豆 (全球) | 15-25% | 地理起源 + 驯化 |
| 拟南芥 (全球 1001 Genomes) | 8-12% | 地理区域（主要是纬度） |
| 棉花 (陆地棉) | 5-10% | 育种群 + 驯化历史 |
| 大麦 (全球) | 15-25% | 春性/冬性 + 地理 + 用途（饲料 vs 啤酒） |

**如何解读 PC1%**：
- 如果 PC1% 远低于物种典型值 → 群体遗传多样性不足。对 GWAS 来说，你只能检测到该群体内分离的位点——如果因果等位基因在该群体中已固定，GWAS 看不到。
- 如果 PC1% 远高于物种典型值 → 可能混入了野生种质、不同亚种或不同育种群的样本。检查元数据。
- PC1% 在种内的变异通常反映：地理隔离 > 育种项目差异 > 市场类型差异。

**PC2, PC3... 的意义**：
- PC2 通常揭示次要的群体分化。对水稻来说，如果 PC1 是 indica/japonica，PC2 通常是地理区域（南亚 vs 东南亚 vs 东亚）。
- 如果 PC2 的解释方差也很大（>10%）且与 PC1 没有数量级的差异 → 群体可能有多个独立的分化轴（如地理 + 冬春性 + 育种群）。
- 前几个 PCs 的解释方差如果总和 < 30% → 群体没有强结构。对于 GWAS，这是好消息——通常只需要 2-3 个 PC 协变量。

## ADMIXTURE

### 选择最佳 K

不要只看 CV error 最低的 K。结合：
1. CV error 的肘点（elbow point）
2. 生物学合理性（已知有多少个品种/群体？）
3. 各 K 值的 bar plot — 是否存在从某个 K 开始出现"杂乱"的混合模式

### 植物中常见的模式

- 育种群体的 ADMIXTURE 通常呈现渐渗模式（introgressed pattern），而非离散簇
- 近交物种的个体可能在某些 K 下显示为多个祖先成分的混合，这不一定是真正的"混合祖先"而是 LD 的影响
- 野生群体更可能呈现离散的祖先成分

### ADMIXTURE 深入：育种群体 vs 野生种质的 K 选择策略

**k 值选择的数学方法**：

1. **交叉验证误差 (CV error)**：运行 ADMIXTURE 的 `--cv` 模式，K 从 1 到 15（或 N_indiv/10 的较小值），选 CV error 最低的 K。这是最常用的方法，但在植物群体中并不总是最优。
   - 对野生群体（离散结构）：CV error 通常有明显的最小值（sharp minimum），K 选择相对明确。
   - 对育种群体（连续结构）：CV error 通常随 K 增加而平稳下降，没有清晰的肘点。此时 CV error 的 K 选择不可靠。

2. **Delta-K 方法 (Evanno method)**：计算相邻 K 值间 Ln P(D) 的二阶变化率。在 Structure 输出中常用，但在 ADMIXTURE 中应用性有限。

3. **Puechmaille 方法 (2016)**：用一个阈值（如 0.8）判断每个个体是否"明确属于"某个祖先成分。选能使最多个体被明确分配且各簇非空的最大 K。

**育种群体的实际 K 选择策略**：

育种群体通常没有离散的结构。你面对的是一条谱系的连续谱 (continuum of relatedness)，而非分离的亚群。在这种情况下的策略：

- 如果 K=2 给出合理的生物学解释（如水稻 indica/japonica，玉米 Stiff Stalk/Non-Stiff Stalk，大豆早熟/晚熟），且后续 K=3+ 变得不可解释——接受 K=2。
- 不要强迫育种材料具有"离散亚群"结构。育种群体确实没有这种结构——PCA 上的连续分布才是真实的模式。
- 如果 GWAS 中只用到前几个 Q 矩阵向量（如 Q1-Q3）作为协变量，那么过大的 K 的部分祖先向量实际上是噪声。K=3 的 Q 矩阵可能比 K=10 的 Q 矩阵更适合作为 GWAS 协变量。
- 报告时诚实说明："ADMIXTURE 分析未揭示离散的亚群结构（所有 K > 2 的个体均呈现高度混合模式），这与该育种群体的连续谱系结构一致。"

**野生种质的 K 选择**：

野生种质通常具有更清晰的结构。在这种情况下：
- CV error 通常可信——选 CV error 最低的 K。
- 结合地理信息验证：K 个簇是否与已知的地理分布一致？
- 如果是全物种范围的野生种质（如水稻的 Oryza sativa + O. rufipogon + O. nivara）→ K 通常 > 5。

## Fst

植物比人类通常有更高的 Fst：
- 野生群体间 Fst 0.2-0.5 正常
- 育种群体间 Fst 0.05-0.15 常见
- 地理隔离群体间 Fst 可达 0.5+

### Fst 深度解读：植物特异阈值

人类群体遗传学中，Fst > 0.15 就被认为是"大"的分化（如欧洲 vs 东亚人群的 Fst ≈ 0.11）。但在植物中，这些阈值需要大幅度上调。

**为什么植物 Fst 比人类高得多**：
1. **近交**：自交物种的 Fst 天然偏高，因为在亚群内，近交进一步固定了等位基因，降低了亚群内的杂合度（H_s ↓ → Fst ↑）。
2. **驯化瓶颈**：作物从野生祖先驯化时经历了强烈的瓶颈效应。不同地区的驯化事件（如 rice 的 indica 和 japonica 独立驯化）产生极高的 Fst。
3. **人工选择**：育种中对不同性状的强烈选择在基因组局部产生极高 Fst 的"选择区域"（selective sweep）。
4. **迁移限制**：作物的种子传播远不如人类的迁移自由。一袋水稻种子从中国传到非洲是近代事件。

**植物 Fst 的解释区间**：

| Fst 范围 | 植物中的人类等价 | 典型场景 |
|----------|---------------|---------|
| < 0.05 | 低分化 | 同一育种项目内不同世代；同一地理区域内的地方品种 |
| 0.05-0.15 | 中等分化 | 不同育种项目间；不同生态型（如春小麦 vs 冬小麦）；相邻地理区域 |
| 0.15-0.30 | 高分化 | 不同种质库类别间（地方品种 vs 现代品种）；远距离地理区域 |
| 0.30-0.50 | 极高分化 | 亚种间（indica vs japonica）；野生近缘种 vs 栽培种 |
| > 0.50 | 物种级分化 | 不同物种（但仍在同一个属内） |

**局部 Fst (window-based Fst) 的特殊价值**：
- 全基因组平均 Fst 告诉你"两个群体总体有多不同"。
- 滑动窗口 Fst 扫描告诉你"哪些基因组区域在群体间差异特别大"。
- 高 Fst 窗口 + 低核苷酸多样性 (π) → 选择性清除 (selective sweep) 的经典信号。这是在植物驯化和育种基因发现中的核心模式。
- 一个高 Fst 窗口如果覆盖了一个已知功能基因（如 sh4 在水稻中控制落粒性，tb1 在玉米中控制分枝）→ 很有可能是育种中的人工选择靶点。

**近交对 Fst 及其他参数的影响**：
- Fst 在自交物种中被**高估**（因为 H_s 被近交压低）。
- 杂合度 (Ho, He) 在自交物种中**被压低**——即使在没有群体结构的情况下，Ho 也远低于 He（这是近交的信号，不是群体结构的信号）。
- 有效群体大小 (Ne) 估计在自交物种中偏小——因为近交降低了有效重组率。

**如何校正**：
- 在计算 Fst 前先过滤掉近交个体（如剔除 Ho < 0.05 或 F > 0.9 的个体，如果群体中有足够多样性的话）。
- 或者按近交系数分层分析——高近交个体单独分组，低近交个体另分一组。
- 如果你想比较自交物种和异交物种的 Fst——不要直接比较数值。数值没有可比性。

## 多倍体感知的群体结构分析

### 为什么标准 PCA 在多倍体中会出错

标准 PCA 假设每个位点是一个二倍体 bi-allelic SNP。在异源多倍体中（如小麦是异源六倍体 AABBDD），这会产生两个层面混淆：

1. **亚基因组间的固定差异被当作群体分化**：小麦 A 基因组上某个 SNP 是 A/A，B 基因组同一位置是 G/G，C 基因组是 G/G。如果一个群体标准的 SNP calling 把它当作 bi-allelic，你会得到了一个 A/G 的"杂合"SNP。"A"等位基因频率在群体 A（A 亚基因组来源多）中更高——但这不是群体分化，是亚基因组差异。
2. **同源多倍体的剂量信号丢失**：同源四倍体马铃薯的一个 SNP 可能有 5 种基因型（AAAA, AAAB, AABB, ABBB, BBBB），对应 0-4 个 B 等位基因。标准 bi-allelic call 只能区分 0/1/2 个 B 等位基因（二倍体框架），丢失了剂量信息。

### 多倍体群体分析的策略

**异源多倍体（小麦、棉花、油菜）**：
- **金标准**：分亚基因组分别分析。将 reads 比对到各亚基因组参考序列，对每个亚基因组独立 call SNP，然后分别做 PCA/ADMIXTURE。
- **如果只能做全基因组分析**：用 polyploid-aware SNP calling（如 polyRec、GATK 的 ploidy 参数）。
- **报告时**：明确说明全基因组 PCA 中的"群体结构"可能部分源于不同品种中亚基因组组成的差异。

**同源多倍体（马铃薯、苜蓿、甘蔗）**：
- 使用专门的 polyploid 分析工具：polyRAD (R/Bioconductor)、GWASpoly 的 PCA 模块、updog 用于基因型 calling。
- 或者将多倍体基因型"降维"为二倍体等价——将等位基因剂量 (0-4) 作为连续变量进行 PCA（这是一种近似但合理的方法）。

## 系统发育树 (NJ)

对植物群体的 NJ 树：
- 近交物种的个体直接聚类（每个品种→明确的簇）
- 异交物种的树通常更"星形"（大量的内部分支）
- 如果树种出现与 PCA 不一致的拓扑结构，优先相信 PCA（NJ 对缺失数据和 rare variants 敏感）

### NJ 树 vs PCA：为什么它们有时不一致，该信哪个

这是植物群体分析中常见的困惑。PCA 和 NJ 树都是基于遗传距离矩阵，但处理距离的方式完全不同。

**为什么会出现不一致**：

1. **NJ 对缺失数据极其敏感**。如果某些个体有大量缺失基因型，它们之间的遗传距离会被高估或低估，导致这些个体在 NJ 树中跑到奇怪的位置。PCA 对缺失数据的稳健性要好得多（通过使用成对完备观测或均值填充）。

2. **NJ 假设进化是树状的（treelike）**。但育种群体有大量的基因流动/渐渗——物种的进化关系是网状的（reticulate），不是树状的。当一个品种是两个远缘祖先的渐渗产物时，NJ 会强行把它放到其中一方，但 PCA 会把它放在两者之间——后者更准确地反映了基因组的混合状态。

3. **稀疏变异 (rare variants)** 在 NJ 中的权重更大。如果某些个体之间存在大量共享的稀有等位基因（可能是因为它们来自同一家系），NJ 会把它们紧密地聚类，即使在全基因组尺度上它们没有那么大差异。PCA 给 common variants 更高的权重。

4. **距离度量的选择**：
   - NJ 依赖特定的距离矩阵（通常是用 Plink `--distance 1-ibs` 或 `--distance matrix`）。IBS 和 IBS 的不同变换（如 sqrt(1-IBS)）会改变树种的结构。
   - PCA 直接对基因型矩阵进行 SVD 分解，不涉及显式的"距离"度量。

**一致性问题决策指南**：

| 情况 | 策略 |
|------|------|
| PCA 看到强 PC1 分化，NJ 树也分开 | 结构可信。同时报告两个结果 |
| PCA 看到强 PC1 分化，NJ 树混在一起 | 检查缺失率。NJ 树受缺失数据影响 |
| PCA 看到连续分布，NJ 树有清晰聚类 | 可能是 rare variants + 系谱效应。报告两个结果并解释差异 |
| PCA 和 NJ 树都不显示强结构 | 群体无强结构。这是一个有效结果 |

**最终建议**：
- PCA 更适合揭示"大尺度"的群体结构（主要分化轴）。
- NJ 树更适合揭示"小尺度"的个体间关系（谁和谁是亲戚）。
- 如果两者矛盾不可调和：对于群体结构判断（GWAS 协变量、亚群划分），**优先相信 PCA**。对于个体级别的亲缘关系（如系谱验证、错误标签检测），NJ 树的信号可能更精细。
- 始终同时展示两个结果，让读者看到分歧的存在。

## 多样性面板的取样策略

如果你在规划一个新的多样性面板（而非分析已有数据）：

### 最大化多样性 vs 最大化 GWAS 功效

这是一个权衡。多样性面板的构成直接决定你发现什么。

**最大化多样性（探索性面板）**：
- 包含该物种的全地理分布范围的种质（地方品种、野生近缘种、育种材料）
- PC1 会非常高（>30%）
- GWAS 会因为严重的群体结构而功效很低——你能看到的分化信号可能是"驯化基因"，因为它们在野生 vs 栽培之间有极大的等位基因频率差异
- 适合：研究驯化、进化、适应性的遗传基础

**最大化 GWAS 功效（育种面板）**：
- 限制在特定的育种群体内（如 elite 育种材料）
- PC1 低（<10%）
- GWAS 功效高——群体结构弱，因果变异和标记之间的 LD 可靠
- 局限性：你只能发现这个育种群体内分离的位点。如果某个抗病基因在这个群体中已经固定了，你检测不到。
- 适合：实际育种中的标记开发

**Nested Association Mapping (NAM) 面板**：
- 多个 founder × 共同亲本的 RIL 群体的集合
- 结合了 bi-parental QTL mapping 的功效和 GWAS 的等位基因多样性
- 玉米 NAM (25 founders × B73 → 5000 RILs) 是经典的植物例子
- 适合：高精度 QTL 定位 + 等位基因系列发现

**MAGIC (Multi-parent Advanced Generation Intercross) 面板**：
- 多个亲本经过多代互交后再自交产生 RILs
- 比 NAM 有更大的重组和等位基因多样性
- 代表性的有：拟南芥 MAGIC (19 founders)、水稻 MAGIC ( indica 8 + japonica 8 founders)

**取样数经验法则**：
- 最少 100 个 accession → 能做 GWAS（低功效）
- 200-300 → 合理
- 400-500 → 良好
- > 1000 → 优良
- 如果是野生种质且结构强 → 每个亚群至少 50-100 个 accession

## 把结果串起来

多个方法交叉验证：
1. PCA 的分群模式 → NJ 树的聚类模式 → ADMIXTURE 的祖先成分 → 三者一致 → 结论可靠
2. 如果不一致：通常是 ADMIXTURE 的 K 选择问题，或数据中存在 close relatives 扭曲了距离矩阵
