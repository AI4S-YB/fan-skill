# 分子标记开发结果解读

## 一句话解释

分子标记开发：将遗传发现（GWAS/QTL 显著位点）转化为实验室可用的检测工具。

## 能回答什么

- 这个 SNP 可以设计成 KASP 标记吗？
- 引物在基因组中是否特异？
- 标记在不同亲本间有多态性吗？

## 不能回答什么

- 标记检测到的等位基因一定与性状有关吗？（取决于原始 GWAS/QTL 发现的可靠性）
- 这个标记在其他群体中有效吗？（取决于 LD 是否保守）
- 功能性标记等同于因果标记吗？（dCAPS 标记的是变异本身但不等同于验证因果）

## 典型输出

| 文件 | 含义 |
|------|------|
| `kasp_primers.csv` | KASP 引物序列和预期基因型 |
| `indel_primers.csv` | InDel 引物和预期产物大小 |
| `primer_specificity.txt` | 引物基因组特异性 BLAST 结果 |
| `parental_alleles.csv` | 各亲本的等位基因状态 |

## 常见结果模式

### 多倍体中引物不特异
在目标亚基因组外有 >80% 同源匹配 → 可能扩增多个亚基因组 → 重新设计。
