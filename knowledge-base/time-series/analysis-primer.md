# 时间序列表达分析结果解读

## 一句话解释

时间序列表达分析：追踪基因表达在多个时间点上的动态变化，识别共表达模块和显著时序模式。

## 能回答什么

- 哪些基因的表达随时间发生了显著变化？
- 有多少个主要的时序表达模式（cluster）？
- 每个 cluster 代表什么生物学过程（GO 富集）？
- 哪些基因在特定组织/阶段特异性高表达（tau 指数）？

## 不能回答什么

- 观察到的时序模式是因果还是伴随现象？（需要突变体或扰动实验验证）
- cluster 内的基因是否是共调控的？（需要启动子 motif 分析 + 实验验证）
- "渐变"模式是真实的渐变还是发育不同步的假象？（需要单细胞或个体水平验证）

## 典型输出

| 文件 | 含义 |
|------|------|
| `cluster_assignments.csv` | 每个基因所属的 cluster 及 membership score |
| `trend_patterns.csv` | STEM 检测的显著时序模式（含统计显著性） |
| `stage_specificity.csv` | 每个基因的 tau 指数（组织/阶段特异性） |
| `cluster_heatmap.pdf` | 各 cluster 的基因表达热图 |
| `line_plots.pdf` | 每个 cluster 的均值时序折线图 |

## 常见结果模式

### 聚类数量多（>15 个 cluster）
可能聚类参数（k）设置过大，或者数据中确实存在丰富的时序模式。先用 elbow/gap statistic 验证最佳 k 值。

### 所有 cluster 都富集到相同的 GO term
可能基因筛选门槛太低（大量低表达无变化基因）。预先过滤低变异基因（如方差前 50%）。

### 昼夜节律 cluster 在胁迫处理后消失
可能不是伪影——处理可能抑制了节律钟。用周期图验证，这是生物学发现而非数据问题。

### 胁迫处理后"0h 和 1h"差异最大但其他时间点无差异
这是急性胁迫响应的典型特征。不是分析问题，而是生物学特征——早期转录爆发的峰很窄。
