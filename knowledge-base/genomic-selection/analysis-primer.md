# 基因组选择结果解读

## 一句话解释

基因组选择 (GS)：用全基因组标记预测每个个体的育种值 (GEBV)，选择 GEBV 最高的个体作为下一代亲本。

## 能回答什么

- 哪些材料育种值最高，应该选做亲本？
- 预测准确性有多高（CV 的 r 值）？
- 选择前 10% 亲本，预期性状提高多少（遗传增益）？

## 不能回答什么

- 哪个基因控制这个性状？（GS 不找因果基因）
- 预测准确性在不同环境中一样吗？（取决于训练环境与目标环境的关系）
- 选出来的亲本后代一定好吗？（GS 预测的是加性效应，显性效应不保证）

## 典型输出

| 文件 | 含义 |
|------|------|
| `gebv_predictions.csv` | 所有个体的预测育种值 |
| `cv_accuracy.csv` | 交叉验证准确性 |
| `gebv_ranking.pdf` | 育种值排序图 |
| `genetic_gain.pdf` | 预期遗传增益 |

## 常见结果模式

### 预测准确性很低 (r < 0.2)
训练群体太小；性状遗传力太低（h²<0.2）；标记密度不足。

### CV 准确性虚高 (> √h²)
说明 CV 策略有问题。随机 k-fold 在结构化群体中会高估。使用 leave-one-family-out CV。

### 模型差异很小
GBLUP vs BayesB 差异通常 <5%。重点优化训练群体设计，而非调模型。
