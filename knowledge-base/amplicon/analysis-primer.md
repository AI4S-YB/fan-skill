# 扩增子测序结果解读

## 一句话解释

扩增子测序：通过 PCR 扩增标记基因（16S/ITS/18S），鉴定样品中的微生物群落组成和多样性。

## 能回答什么

- 样品中有哪些细菌/真菌？
- 不同处理组间的群落结构是否不同？
- 哪些微生物类群在不同组间差异显著？

## 不能回答什么

- 微生物具体在做什么功能？（需要宏基因组）
- 微生物是活的还是死的？（需要 RNA 或 PMA 处理）
- 绝对丰度是多少？（扩增子是相对定量）

## 典型输出

| 文件 | 含义 |
|------|------|
| `asv_table.csv` | ASV/OTU 在各样本中的丰度 |
| `taxonomy_table.csv` | 每个 ASV 的分类学注释 |
| `alpha_diversity.csv` | Shannon、Chao1 等多样性指数 |
| `pcoa_plot.pdf` | Beta 多样性排序图 |

## 常见结果模式

### 植物 DNA 污染严重 (>50% reads 是叶绿体/线粒体)
在根内圈和叶圈样本中完全正常。使用 PNA 夹阻断或生物信息学过滤去除。

### 处理组和对照无显著差异
可能原因：生物重复不够（建议 n≥6）、效应本身小、取样时间窗口不对。

### 稀有 ASV 过多
过滤掉总丰度 <0.01% 或出现频率 <5% 的 ASV。但不要过度过滤。
