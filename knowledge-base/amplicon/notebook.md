# 植物扩增子测序：分析笔记

这份笔记帮助你像一个有经验的微生物生态学家一样思考和做决策。

## 在开始之前：理解你的实验设计

扩增子分析的成功与失败，很大程度上在实验设计阶段就已决定。

### 选择什么标记基因？

| 目标类群 | 标记基因 | 可变区 | 数据库 | 分辨率 |
|----------|----------|--------|--------|--------|
| 细菌/古菌 | 16S rRNA | V3-V4 或 V4 | SILVA 138 | 属水平为主 |
| 真菌 | ITS | ITS1 或 ITS2 | UNITE | 种水平可能 |
| 真核微生物 | 18S rRNA | V4 或 V9 | PR2 / SILVA | 科-属水平 |
| 固氮菌 | nifH | - | 专用数据库 | 功能群水平 |
| 氨氧化菌 | amoA | - | 专用数据库 | 功能群水平 |

植物微生物组研究最常用的是 16S V4 区（细菌）和 ITS1 区（真菌）。

### 样本类型

植物微生物组样本类型和特点：

- **根际土壤（Rhizosphere）**：微生物量最大，多样性最高。取样时注意附着在根表的土壤厚度影响结果。通常细菌负荷 10^8-10^9/g 土壤。

- **根内圈（Endosphere）**：根内微生物，经过植物免疫筛选的"精选"群落。生物量低，植物 DNA 污染是主要问题。PCR 循环数可能需要增加，但要警惕外源污染。

- **叶圈（Phyllosphere）**：叶表微生物，受 UV、干燥、营养贫乏影响。生物量极低，通常只做 16S。阴性对照极其重要。

- **种子内生菌（Seed endophyte）**：生物量极低。需要严格的去污染操作。建议用 PNA 或 LNAs 嵌段剂减少植物 DNA 干扰。

### 生物量问题

如果你分析的是根内圈或叶圈样本，植物 DNA 占 90%+ 是正常的。这意味着：
- 需增加 PCR 循环数（但要记录，因为会引入扩增偏好）
- 需设置多个阴性对照（DNA 提取空白、PCR 空白）
- 化学污染假说：低生物量样本中常见的外源 DNA 污染来自试剂盒本身
- 考虑 PNA 夹特异性阻断植物线粒体和叶绿体 16S 扩增

## 选择你的去噪策略

### DADA2：为什么是最优选择

DADA2 产生的是 amplicon sequence variant（ASV），分辨率达到单核苷酸水平。

**优势**：
- 比 OTU（97% 聚类）的分辨率高
- 可跨研究比较（序列本身是标识符）
- 内置嵌合体检测和质量过滤

**局限**：
- 需要足够的计算资源（大型数据集可能内存不足）
- 对测序错误敏感——需要足够覆盖深度来区分生物学变异和测序错误

### Deblur vs DADA2

- Deblur 要求固定读长，适合所有 reads 等长的情况（如 V4 区 250bp PE 合并后恰好为固定长度）
- DADA2 更灵活，可以处理可变读长
- 如果数据质量好且读长固定，两者差别不大
- 如果有读数被截断（如 V3-V4 长片段），DADA2 更好

### 什么时候考虑 OTU？

如果测序质量极差（如早期的 454 数据），97% OTU 聚类（VSEARCH/QIIME1）可能比 ASV 法更稳健。但对于现代 Illumina 数据，ASV 方法已经是标准。

## 分类学注释：数据库选择

### 16S rRNA → SILVA 138

SILVA 是植物微生物组的标准选择：
- 更新频率稳定（每 1-2 年）
- 涵盖植物相关细菌类群全面
- 叶绿体和线粒体序列包含在内→可用于识别和去除植物来源读数

**替代方案**：
- **Greengenes2**：如果关注人类相关的比较或使用 QIIME 2 新插件
- **GTDB (Genome Taxonomy Database)**：如果想用基于基因组的分类体系

### ITS → UNITE

UNITE 是真菌 ITS 注释的唯一首选：
- 包含 10 万+ 真菌物种的参考序列
- 动态分类（专家策划的物种假设）
- 支持多种分类置信度阈值

植物病原真菌在 UNITE 中覆盖良好（镰刀菌、链格孢、锈菌等）。

### 分类置信度阈值

- 门水平 > 0.8
- 纲水平 > 0.8
- 属水平 > 0.7（16S 通常难以到种）
- ITS 到种水平的置信度因类群不同而异

## 多样性分析：必要 vs 可选

### 必须做的

**Alpha 多样性**：
- Shannon 指数（兼顾丰富度和均匀度）
- Observed features（ASV/OTU 数量）
- Faith's PD（系统发育多样性，需要建树）

**Beta 多样性**：
- Bray-Curtis 距离（丰度加权，不依赖系统发育）
- Weighted/Unweighted UniFrac（依赖系统发育，揭示不同层面的群落差异）

**排序方法**：
- PCoA（主坐标分析）用于可视化

### 可选的

- Simpson 指数（关注优势种）
- Jaccard 距离（仅基于存在/缺失）
- Aitchison 距离（基于 CLR 变换，稳健）

---

## 差异丰度分析：不要盲目用 LEfSe

LEfSe 曾经是最常用的工具，但已暴露严重的假阳性问题：

1. **ANCOM-BC**（当前最推荐）：控制 FDR，对稀疏数据稳健，校正了测序深度偏差
2. **ALDEx2**：基于 CLR 变换，贝叶斯方法，对小样本稳健
3. **DESeq2**（谨慎使用）：设计用于 RNA-seq，需要适当的归一化
4. **LEfSe**：不推荐作为主要方法，可用作探索性分析

## 常见陷阱

### 植物 DNA 污染

植物线粒体 16S 和叶绿体 16S 与细菌 16S 高度同源，容易被扩增。对策：
- PNA 夹阻断植物序列（推荐用于根内圈和叶圈）
- 生物信息学移除：比对 SILVA 后过滤掉注释为 "Chloroplast" 和 "Mitochondria" 的 ASV
- 使用植物特异性引物（799F/1193R 可减少叶绿体扩增）

### 批次效应

微生物组数据极易受批次效应影响：
- DNA 提取批次是关键混杂因素
- 将提取批次的 ID 加入统计模型
- 如果阳性和阴性对照在同一批次中表现异常，该批次所有样本可能都受影响

### 稀有 ASV

低丰度 ASV（如总读数 < 10 或出现在 < 2 个样本中）可能是：
- 真实的稀有类群
- 测序错误（dada2 的去噪后的残留错误）
- 交叉污染

通常建议过滤掉总丰度低于 0.01% 或出现在少于 5% 样本中的 ASV。但不要过度过滤。

### 组成效应

扩增子数据是相对的（百分比），不是绝对的：
- 一个类群的升高必然导致其他类群的降低（即使其他类群实际并未变化）
- 这是所有基于扩增子的微生物组研究的根本限制
- ANCOM-BC 部分克服了这个问题，但不能完全消除
- 如果要得到绝对丰度，结合 qPCR 或流式细胞计数（spike-in 方法）

## 16S vs ITS vs 18S：植物微生物组的标记基因选择

植物微生物组研究中，不同标记基因对应不同的生态位和科学问题。

### 按生态位选择

| 生态位 | 推荐标记基因 | 理由 |
|--------|------------|------|
| 根际土壤细菌 | 16S V4 或 V3-V4 | 细菌多样性高，V4 区分辨率足够 |
| 根际土壤真菌 | ITS2 | 真菌多样性最高在此生态位 |
| 根内圈细菌 | 16S V4 + PNA 夹 | 需抑制植物线粒体/叶绿体扩增 |
| 根内圈真菌 | ITS2 | 丛枝菌根真菌(AMF)在 ITS2 中覆盖好 |
| 叶圈细菌 | 16S V4 或 V5-V7 | 叶圈细菌群落结构简单，V4 足够 |
| 叶圈真菌 | ITS1 或 ITS2 | 低生物量，需加大 PCR 循环数 |
| 种子内生菌 | 16S V4 + ITS2 | 极低生物量，需多重阴性对照 |

### 按科学问题选择

- **细菌 + 真菌群落结构** → 16S V4 + ITS2（最常规组合）
- **功能群特定问题** → 16S V4 + 功能基因（nifH/amoA/nirK）
- **真核微生物多样性** → 18S V4 或 V9（仅需要时）
- **土壤食物网初级消费者（线虫、原生生物）** → 18S V9
- **AMF 菌根真菌** → 18S 或专门的 SSU-ITS2-LSU 长片段

### 18S 在植物微生物组中的特殊用途

18S 通常不用于细菌和真菌（16S 和 ITS 分辨率更好），但在以下场景中有价值：
- 土壤原生生物多样性评估（对土壤养分循环和细菌捕食有重要作用）
- 线虫群落分析（植物寄生线虫 vs 自由生活线虫）
- 藻类和卵菌检测（植物病原卵菌如疫霉属可用 18S 或专门的 COX 标记）

## 土壤 vs 根 vs 叶 vs 种子：植物微生物组采样策略

不同生态位的采样策略差异巨大，直接影响下游结果。

### 根际土壤采样

**什么是根际土壤**：附着在根表 1-5mm 的土壤，受根分泌物影响。
- **取样方法**：将植株连根挖出，轻轻抖落松散土壤（非根际土），用无菌刷或刮刀收集紧附在根表的土壤（根际土）
- **根系分泌物影响**：不同植物、不同生长阶段根分泌物差异大→微生物群落随之变化
- **土壤异质性**：同一田块的不同位置根际微生物可能不同 → 混合多点取样
- **最少取样量**：0.5-2g 根际土壤用于 DNA 提取（取决于土壤有机质含量）
- **保存**：液氮速冻后 -80C 保存；或使用 RNAlater/DNA 稳定剂。不要在室温下存放超过 2 小时

### 根内圈采样

- **表面消毒**：次氯酸钠 + 乙醇梯度消毒，DM 超声去除表面附着微生物
- **消毒效果验证**：将最后一次洗涤液 PCR 作为阴性对照。如果洗液 PCR 阳性，消毒不充分
- **根组织量**：不同植物差异大 — 拟南芥需要上百株，玉米 3-5 株即可
- **时间敏感**：根离体后微生物群落在 30 分钟内开始变化 → 采集后立即处理

### 叶圈采样

- **取样策略**：取完全展开的健康叶片，避免老叶和病叶（除非研究目标是病害）
- **叶龄效应**：幼叶和成熟叶的微生物群落不同 → 固定叶位取样
- **环境因子**：雨后 48 小时内不要取样（叶表微生物被冲刷）；记录 UV 指数
- **可培养 vs 总 DNA**：叶圈总 DNA 包含活菌和死菌。如需活性群落信息，用 PMA 处理区分活/死菌

### 种子内生菌采样

- **表面消毒更加严格**：种子表面消毒方案需验证（洗液 PCR 阴性才能确认内部微生物）
- **样本量**：小种子（拟南芥、烟草）可能需要数千粒；大种子（大豆、玉米）50-100 粒
- **CTAB 或专门试剂盒**：种子 DNA 提取使用 CTAB 法或含有裂解困难细菌的 bead-beating 步骤的试剂盒
- **多重阴性对照至关重要**：提取空白、PCR 空白、环境对照（打开的空管暴露在操作环境中）

### 采样策略综合建议

| 研究目标 | 推荐生态位 | 最少重复数 | 关键对照 |
|---------|----------|----------|---------|
| 微生物组组成描述 | 根际 + 根内 + 叶 | 每生态位 n=5-8 | 阴性对照 |
| 干旱/盐胁迫响应 | 根际 + 根内 | 每处理 n=6-10 | 环境样本 + 阴性对照 |
| 病害抑制 | 根际 + 根内 | 每处理 n=6-8 | 健康对照 + 不种植物土壤 |
| 施肥效应 | 根际 + 非根际土壤 | 每处理 n=5-8 | 不施肥对照 + 土壤 baseline |
| 种子垂直传播 | 种子 + 幼苗根/叶 | 每代 n=6-10 | 母本植物 + 空白提取 |

## 农业场景下的多样性解读

植物微生物组研究中，alpha 和 beta 多样性的变化需要结合农业背景解读。

### Alpha 多样性与农艺处理

- **施肥效应**：长期施用化肥通常降低细菌 alpha 多样性（喜肥类群如变形菌门富集）。有机肥则可能增加多样性（引入外源微生物 + 改善土壤有机质）。短期施肥效应可能不明显。
- **干旱**：中度干旱可能不影响或增加根际多样性（根分泌物增多招募多种微生物），严重干旱降低多样性。叶圈多样性通常随干旱下降。
- **病害**：土传病害爆发通常伴随根际多样性降低（病原菌占主导），但某些抑病型土壤中高多样性与病害抑制相关。
- **轮作/间作**：通常增加根际多样性。作物种类越多样，微生物多样性越高。

### Beta 多样性与农艺处理

- **施肥改变群落结构**：化肥 vs 有机肥的群落结构差异通常在 PCoA 中沿 PC1 分离。化肥处理下群落结构更均质（样本间距离小）。
- **干旱梯度**：干旱强度与群落结构变化通常呈线性关系 — 可量化为 Bray-Curtis 距离与土壤水势的相关性。
- **基因型效应**：不同品种的根际微生物组差异（品种效应）通常小于处理效应（如施肥、灌溉），但某些品种（如抗病品种）可能有显著不同的微生物组。
- **时间效应**：生长季节中的微生物群落演变（temporal turnover）通常大于短期处理效应。长期时间序列实验需考虑取样时间与处理效应的交互作用。

### 网络分析在农业微生物组中的解读

- **共现网络（Co-occurrence network）**：在胁迫条件下网络复杂度降低（节点数和边数减少）是常见的指标
- **关键物种（Keystone taxa）**：网络中的 hub 节点可能代表生态功能关键物种。注意：基于相关性的 hub 不一定是因果关系
- **正/负相关比例**：环境胁迫下负相关比例升高可能反映竞争加剧

## 常见陷阱补充

### 叶绿体和线粒体交叉扩增

植物 16S 扩增子数据中，叶绿体和线粒体来源的序列可占总读数的 50-90%（取决于样本类型和引物选择）。

**机制**：细菌 16S rRNA、植物线粒体 16S rRNA 和叶绿体 16S rRNA 均来源于同一祖先（内共生起源），序列高度保守。通用引物如 515F/806R (V4) 和 341F/805R (V3-V4) 对植物细胞器 16S 都有一定程度的交叉扩增。

**缓解策略**（按有效性排序）：
1. **PNA 夹**（最有效）：在 PCR 中加入与植物线粒体/叶绿体 16S 互补的 PNA（peptide nucleic acid）探针。PNA 与植物序列结合后阻止 DNA 聚合酶延伸。适用于根内圈和叶圈样本。商业 PNA 夹如 pPNA（阻断叶绿体 16S）和 mPNA（阻断线粒体 16S）
2. **植物特异性引物**：799F/1193R 可减少叶绿体扩增（但仍会扩增部分蓝细菌）。799F/1392R 对植物细胞器 16S 的交叉扩增最小
3. **生物信息学过滤**：比对 SILVA 后识别并移除注释为"Chloroplast"和"Mitochondria"的 ASV。注意 SILVA 中植物细胞器 16S 注释可能不完整，部分低丰度植物序列可能仍被误标为细菌
4. **CRISPR-Cas9 靶向去除**（新兴方法）：使用 Cas9 和植物细胞器特异性 gRNA 在文库构建前切割植物序列

### 低生物量样本的特殊处理

根内圈、叶圈和种子样本属于低微生物生物量样本（与土壤相比生物量低 100-1000 倍）：

**污染问题**：
- 试剂盒本身含有微量细菌 DNA（"kitome"）。在低生物量样本中，kitome 可占测序读数的 30-80%
- 实验室空气中的细菌 DNA（灰尘、皮屑）是另一个污染源
- 污染通常以 α-变形菌纲（如 Sphingomonas、Bradyrhizobium）和 β-变形菌纲（如 Ralstonia）为代表

**污染控制策略**：
1. 记录每种试剂和耗材的批次号
2. DNA 提取空白对照（每个提取批次 1-2 个）
3. PCR 空白对照（每个 PCR 板 1 个）
4. 如果污染 ASV 在空白对照中丰度高于实际样本 → 使用 `decontam` R 包移除
5. 使用 UV 或 DNase 处理工作台和移液器
6. 如果可能，使用单细胞或低输入量建库方法
