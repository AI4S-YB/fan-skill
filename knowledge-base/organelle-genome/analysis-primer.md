# 细胞器基因组分析结果解读

## 一句话解释

细胞器基因组分析：组装和注释叶绿体/线粒体基因组，研究其结构、进化及应用价值。

## 能回答什么

- 细胞器基因组的完整序列是什么？
- 包含哪些基因？
- IR边界在哪里？
- 与其他物种的关系如何？
- 是否有结构变异？

## 不能回答什么

- 细胞器基因的功能验证？（需要实验）
- 核-细胞器互作？（需要额外数据）
- 异质体比例？（需要专门分析）

## 典型输出

| 文件 | 含义 |
|------|------|
| `assembly.fasta` | 组装序列 |
| `annotation.gb` | GenBank格式注释 |
| `genome_map.pdf` | 基因组环形图 |
| `ir_boundary.pdf` | IR边界图 |
| `comparison.svg` | 比较分析图 |
| `phylogeny.treefile` | 系统发育树 |

## 常见结果模式

### 叶绿体典型结构

四分结构清晰可见：LSC-IRb-SSC-IRa。基因排列高度保守，可用于系统发育和条形码研究。

### IR边界变异

IR边界的扩张/收缩是叶绿体基因组变异的主要来源，可用于区分近缘物种。

### 基因丢失

某些谱系可能丢失特定基因（如accD、ycf1、ycf2），这是进化的标志。

### 线粒体复杂性

植物线粒体基因组可能显示复杂的重组模式和RNA编辑位点。

## 植物特异性结果解读

### 叶绿体保守性

叶绿体基因组在大多数植物中高度保守，适合用于科级及以上分类单元的系统发育分析。

### IR扩张/收缩

IR边界的变异是叶绿体基因组长度差异的主要原因，某些类群有明显的IR扩张或收缩。

### 线粒体多样性

植物线粒体基因组大小和结构变异极大，反映了不同的进化历史和生活方式。

### RNA编辑

植物线粒体和叶绿体基因组存在大量RNA编辑位点（主要是C→U），影响基因注释。

## 结果报告建议

1. 报告基因组大小和GC含量
2. 展示基因组成环形图
3. 分析IR边界位置
4. 统计基因类型和数量
5. 如有比较分析，展示比对结果
6. 讨论结构变异和进化意义
