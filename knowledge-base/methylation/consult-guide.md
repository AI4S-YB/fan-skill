# DNA 甲基化分析引导

## 这个分析解决什么问题

用户想分析全基因组 DNA 甲基化模式。典型表述：
- "处理前后甲基化有什么变化"
- "哪些区域发生了差异甲基化 (DMR)"
- "甲基化和基因表达有什么关系"

## 你需要问什么

1. **测序策略**: WGBS (全基因组)还是 RRBS (简化)？
2. **物种和参考基因组**: 植物甲基化有三种上下文 (CG/CHG/CHH)
3. **实验设计**: 处理/对照？每组几个重复？
4. **分析目标**: 甲基化水平定量 / DMR / 与RNA-seq关联？
5. **数据量**: WGBS 建议 ≥15x 覆盖度

## 植物特定考量

- **三种 context**: 植物 DNA 甲基化发生在 CG、CHG 和 CHH 三种序列 context 中，由不同的甲基转移酶维持（MET1、CMT3、DRM2/CMT2）。必须分别分析，不能合并
- **WGBS vs RRBS**: 全基因组亚硫酸氢盐测序（WGBS）是金标准；简化代表性亚硫酸氢盐测序（RRBS）成本低但偏向 CpG 岛——植物中 CHG/CHH 甲基化会被遗漏
- **转座子甲基化**: 植物转座子（TE）通常高度甲基化（CHG 和 CHH context）。TE 甲基化状态与附近基因的表达密切相关
- **组织特异性**: DNA 甲基化在植物不同组织中存在差异（如胚乳低甲基化、花粉中 siRNA 介导的甲基化）。明确样本组织是关键

## 对话注意事项

- 使用生物学语言而非工具名称解释推荐理由（如"这个分析找控制性状的基因组区域"而非"用 GAPIT 跑混合线性模型"）
- 一次只问一个问题，根据用户的回答自然推进对话
- 如果用户已提供的信息覆盖了上述某个问题，跳过不要重问
- 确保在推荐分析前收集到足够信息——信息不足时推荐不准确
