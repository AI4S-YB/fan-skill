# RNA-seq 差异表达结果解读

## 一句话解释

找处理组和对照组之间表达发生显著变化的基因。

## 能回答什么

- 处理引起了哪些基因的表达变化？
- 变化了多少倍？显著性如何？
- 这些变化涉及哪些生物学通路？

## 不能回答什么

- 表达变化是原因还是结果？（相关性 ≠ 因果）
- 蛋白水平也变了吗？（mRNA ≠ 蛋白）

## 典型输出

| 文件 | 含义 |
|------|------|
| `differential_expression_table.csv` | 每个基因的 log2FC、p 值、FDR |
| `volcano_plot.pdf` | 火山图：横轴 fold change，纵轴显著性 |
| `heatmap.pdf` | 热图：Top DEGs 的表达模式 |

## 常见结果模式

### DEG 数量异常多 (>30%)
可能原因：处理剂量太高、存在未校正的批次效应、发育阶段不同步。
→ 先看 PCA 图确认分组是否按处理/对照分离。

### 没有 DEGs
可能原因：功效太低（重复不够）、处理确实无影响、表型测量误差。
→ 2 重复时这是常见的。增加重复可以改善。

### 非模式物种 GO 富集为空
不是"无通路差异"。是注释不完整。
→ 用 Mercator (https://www.plabipd.de/portal/mercator4) 在线注释后再做 GO。
