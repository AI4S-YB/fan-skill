# 变异检测结果解读

## 一句话解释

变异检测：将重测序 reads 比对到参考基因组，鉴定样本与参考之间的 SNP、InDel 和结构变异。

## 能回答什么

- 样本中有多少个 SNP/InDel？
- 变异在基因组上的分布如何（基因间区、外显子、内含子）？
- 每个样本的杂合度是否正常？
- SNP 集的质量如何（Ti/Tv、缺失率）？

## 不能回答什么

- 鉴定到的 SNP 与表型有关吗？（需要 GWAS/QTL 分析）
- 这个 SNP 是功能性的吗？（需要 SnpEff/VEP 注释 + 实验验证）
- 多倍体中每个同源等位基因的准确剂量？（同源多倍体的同源染色体无法区分）

## 典型输出

| 文件 | 含义 |
|------|------|
| `variants.vcf.gz` | 变异检出结果（过滤后） |
| `variant_stats.csv` | 变异统计（总数、Ti/Tv、杂合度、缺失率） |
| `qc_report.html` | 质控报告（覆盖度分布、变异密度图） |

## 常见结果模式

### Ti/Tv < 1.5
假阳性过多。可能原因：测序错误过多、参考和样本亲缘关系太远、或包含了重复区域的假变异。检查 FastQC 和比对率。

### 自交物种杂合度 >5%
可能是样本污染/混样、参考基因组错误导致的假杂合、或该"品种"实际是混合材料。用 Picard MarkDuplicates 去重复后重新检查。

### 着丝粒区域变异密度骤降
正常现象——着丝粒富含重复序列，reads 难以唯一比对。这不是数据质量问题，报告中注明即可。

### 细胞器基因组覆盖度极高
叶绿体和线粒体 DNA 占总 DNA 的 5-20%，覆盖度通常是核基因组的 50-200x。应单独分析细胞器变异，不要混入核基因组 SNP 集中。
