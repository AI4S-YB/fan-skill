# 蛋白质组学结果解读

## 一句话解释

蛋白质组学：系统地鉴定和定量生物样品中的全部蛋白质。

## 能回答什么

- 实验条件下哪些蛋白丰度发生了显著变化？
- 差异蛋白富集在哪些功能通路？
- 蛋白-蛋白互作网络中有哪些 hub 蛋白？

## 不能回答什么

- 蛋白丰度变化一定对应功能变化吗？（翻译后修饰可能更重要）
- mRNA 和蛋白的相关性为什么弱？（植物中 r=0.4-0.6 正常）
- 所有蛋白都被检测到了吗？（取决于丰度、溶解度和质谱灵敏度）

## 典型输出

| 文件 | 含义 |
|------|------|
| `protein_groups.csv` | 鉴定和定量的蛋白组 |
| `differential_proteins.csv` | 显著差异蛋白 |
| `pathway_enrichment.csv` | 差异蛋白的通路富集 |
| `ppi_network.csv` | 蛋白互作网络 |

## 常见结果模式

### Missing values 高 (DDA 中 10-30%)
DDA 固有的随机前体选择问题。用 DIA 可降到 <5%。对已有 missing values：MaxQuant 的 match-between-runs 可部分恢复。

### mRNA-蛋白相关性弱 (r < 0.4)
植物中正常。翻译后调控、蛋白降解等贡献了差异。

### 非模式物种蛋白鉴定少
用 RNA-seq 转录本构建样本特异性蛋白库，不要只用近缘物种的参考蛋白质组。
