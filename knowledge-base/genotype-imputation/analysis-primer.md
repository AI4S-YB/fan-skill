# 基因型填充结果解读

## 一句话解释

基因型填充（Imputation）：利用 LD 信息，将低密度基因型数据推断到高密度。

## 能回答什么

- 填充后的标记密度提高了多少？
- 哪些标记填充质量高（R² > 0.8），可以直接用于 GWAS？
- 低频变异的填充精度如何？

## 不能回答什么

- 填充的基因型是 100% 正确的吗？（需要实测验证）
- 填充精度在所有基因组区域都一样吗？（着丝粒区域精度很差）
- 不同填充工具的结果一致吗？（需要交叉验证）

## 典型输出

| 文件 | 含义 |
|------|------|
| `imputed_genotypes.vcf.gz` | 填充后的基因型 |
| `imputation_quality.csv` | 每个 SNP 的 R²/INFO 评分 |
| `pre_post_comparison.csv` | 填充前后标记数对比 |

## 常见结果模式

### 低频变异 (MAF<0.01) 填充精度低
正常。保守地过滤 R²<0.5 的低频变异。

### 参考面板与目标群体不匹配
PCA 检查：如果参考和目标在 PC1 分离明显 → Fst 较大 → 填充精度会下降。筛选与目标遗传距离近的参考个体。

### 多倍体中填充精度低
分亚基因组独立填充比全基因组联合填充好 5-15% R²。
