# 环境组学结果解读

## 一句话解释

环境组学（Enviromics）：量化环境特征，分析基因型×环境互作（G×E），为不同环境类型推荐最优品种。

## 能回答什么

- 哪些基因型在不同环境中表现最稳定？
- G×E 效应在总表型变异中占多大比例？
- 测试环境可以分为几个"目标环境群体 (TPE)"？
- 哪些气候变量最影响品种表现？

## 不能回答什么

- 未来气候下品种表现如何？（需要气候情景模拟，有不确定性）
- TPE 划分是永久不变的吗？（气候变化会改变 TPE 边界）
- 环境间差异一定能被解释吗？（有些 G×E 不能用现有环境变量解释）

## 典型输出

| 文件 | 含义 |
|------|------|
| `ammi_biplot.pdf` | AMMI/GGE 双标图：基因型×环境关系 |
| `stability_indices.csv` | 各基因型的稳定性参数 |
| `tpe_clusters.csv` | TPE 聚类分组结果 |
| `reaction_norms.pdf` | 反应规范图：基因型对环境梯度的响应 |

## 常见结果模式

### G×E 方差 > 基因型方差
品种推荐必须分环境类型。不同 TPE 需要不同的品种。

### 少数环境驱动了大部分 G×E
检查异常环境（极端天气年份）。如果移除该环境后 G×E 不显著→G×E 主要由异常环境驱动。
