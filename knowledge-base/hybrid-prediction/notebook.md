# 植物杂交预测分析：分析笔记

## 在开始之前

杂交预测（Hybrid Prediction）是植物育种中最有经济价值但最复杂的分析方法之一。核心目标：在种植杂交种之前，预测哪些组合会产生最佳表现。先问自己：

### 你有什么样的数据？

- **NCII 设计 (North Carolina Design II)**：A 组 × B 组（两套测验杂交系）
- **双列杂交 (Diallel)**：所有亲本两两杂交（完整双列或半双列）
- **因子设计 (Factorial)**：类似 NCII 但可能不完全
- **只有历史杂交数据**：多年多点历史试验数据

### 亲本信息

- **基因型数据**：SNP/DArT 标记 → 可做基因组预测
- **系谱 (Pedigree)**：≥3 代人系谱 → 可估计育种值
- **只有表型**：配合力数据 → 只能做 GCA-based 预测

### 你的育种目标是什么？

- **筛选最佳杂交组合** → 基因组预测 + 排名
- **划分杂优群** → 杂种优势模式分析
- **预测未测组合** → 基因组 BLUP 或 BayesR
- **优化育种方案** → 最优贡献选择

## 配合力分析 (Combining Ability)

### GCA (一般配合力) 和 SCA (特殊配合力)

- **GCA**：某个亲本在所有杂交组合中的平均表现 → 代表加性遗传效应
- **SCA**：特定亲本组合偏离 GCA 期望的偏差 → 代表非加性效应（显性+上位性）

### NCII 设计分析

最常见的水稻、玉米杂交育种设计：

```
yield = mu + GCA_male[i] + GCA_female[j] + SCA[i,j] + error

其中:
- GCA 效应 ~ N(0, sigma_GCA^2)
- SCA 效应 ~ N(0, sigma_SCA^2)
```

**关键比率**：Baker's ratio (GCA 方差与总遗传方差之比)
- >0.8 → 杂交表现主要由 GCA 决定 → 可以用亲本表现预测杂交表现
- 0.5-0.8 → GCA 和 SCA 都重要
- <0.5 → SCA 占主导 → 特定组合效应非常重要

### 双列杂交设计 (Diallel)

完整双列杂交：所有亲本两两杂交（含正反交）。

**四种 Griffing 方法**：
- **方法 1**：包含亲本 + 正反交 F1（最完整，但实验规模最大）
- **方法 2**：亲本 + 仅正交 F1（无正反交）
- **方法 3**：仅正交 F1（无亲本、无正反交）
- **方法 4**：仅正交 F1（与方法 3 区别在于固定 vs 随机效应的假设）

**植物常用**：方法 2 或方法 4。方法 1 在植物中很少用（正反交效应在大多数作物中不显著）。

**规模计算**：
- n 个亲本的完整双列杂交 = n(n-1)/2 个杂交组合（方法 4）
- 10 个亲本 → 45 个组合
- 50 个亲本 → 1225 个组合（快速超出实验能力）

### 因子设计 (Factorial)

A 组 × B 组交叉，但不是所有组合都做。

**适合**：当 A 组是候选亲本而 B 组是标准测验种时。
**局限**：某些 A × B 组合缺失 → 对应亲本的 GCA 估计不精确。

### 植物特异性

- **水稻**：GCA 通常占主导（特别是产量性状），SCA 较小 → 选择优良亲本即可
- **玉米**：GCA 和 SCA 都重要 → 需要同时考虑亲本表现和特定组合
- **油菜**：杂种优势强，SCA 往往占更高比例
- **小麦**：杂交小麦起步较晚，配合力模式与其他作物有所不同

## 杂优群划分 (Heterotic Grouping)

### 为什么需要杂优群？

不同杂优群之间的杂交通常产生更强的杂种优势。相反，同一杂优群内的杂交 → 杂种优势弱。

### 分子标记 vs 配合力：两种划分策略

**策略 A：分子标记 (Molecular markers)**

基于全基因组 SNP 数据：

1. **遗传距离 (IBS/IBS)**：计算所有亲本间的遗传距离矩阵
2. **PCA**：前几个主成分通常能清晰分离杂优群
3. **Structure/Admixture**：基于模型的群体结构分析
4. **系统发育树 (NJ)**：对亲本聚类

**优点**：不依赖表型数据，客观，可重复
**缺点**：遗传距离和杂种优势的关系在不同作物中可能复杂（存在"最佳遗传距离"而非线性关系）

**策略 B：配合力划分 (Combining ability-based)**

基于 GCA 和 SCA 估计值：
1. 计算所有亲本的 GCA 效应
2. 对 GCA × GCA 矩阵做聚类
3. 识别在杂交中表现互补的亲本群体

**优点**：直接基于育种价值，与杂种优势直接相关
**缺点**：需要已经有一部分杂交表型数据（无法用于全新的亲本组）

**推荐做法**：先用分子标记做初步划分 → 用配合力数据验证和优化划分

### 经典杂优群 (举例)

| 作物 | 典型杂优群 |
|------|-----------|
| 玉米 | Stiff Stalk × Non-Stiff Stalk; Iowa Dent × Lancaster |
| 水稻 | indica × japonica; 两系 × 恢复系 |
| 油菜 | 冬油菜 × 春油菜; 不同来源群体 |
| 向日葵 | 保持系 × 恢复系 |

### 非模式物种怎么办？

- 如果没有已知的杂优群：先用全基因组 PCA 聚类，找出潜在群体结构
- 验证：将潜在群体作为杂交设计的"组"因子，检查组间杂交是否确实表现更好

## 基因组预测 (Genomic Prediction)

### 基因组预测 vs 系谱预测

| 方法 | 信息来源 | 优点 | 缺点 |
|------|---------|------|------|
| GBLUP (基因组) | 全基因组 SNP | 可预测无系谱记录的亲本；可预估孟德尔抽样 | 需要基因型数据；标记密度不足时精度低 |
| ABLUP (系谱) | 系谱关系矩阵 (A-matrix) | 无需基因型数据；计算快 | 只能用于系谱记录完整的群体；忽略孟德尔抽样 |
| HBLUP (混合) | G + A 组合 | 整合两种信息源；精度最高 | 需要两种数据；模型更复杂 |

### 杂交预测的特殊建模挑战

**必须在模型中包含显性效应 (Dominance)**：

标准 GBLUP（仅加性效应）：
```
y = mu + Z_a * a + e
```

杂交 GBLUP（加性+显性）：
```
y = mu + Z_a * a + Z_d * d + e

其中 d 的协方差由显性关系矩阵 D 定义
```

**原因**：杂交中的显性效应（杂合子效应）是杂种优势的主要遗传基础。忽略显性效应会严重低估杂交预测准确性（通常降低 0.1-0.2 Pearson r）。

### GBLUP vs BayesR

- **GBLUP**：假设所有标记效应相同 → 快速，适合多性状
- **BayesR**：标记效应有不同的方差 → 更准确但更慢，适合高密度标记
- **BayesB**：大部分标记效应为 0 → 适合只有少数 QTL 控制性状的情况

### 植物杂交预测的特殊考虑

- **加性 vs 显性效应**：在模型中包含显性效应（Dominance）通常可以提高杂交预测准确性
- **G×E 互作**：多年多点数据的杂交预测需要考虑基因型×环境互作
- **亲本信息**：如果亲本本身也被评价，可以用亲本表型作为协变量

### 预测未测杂交组合

```
模型: yield ~ GCA_male + GCA_female + SCA

训练集: 已测杂交组合 (如 100 个 F1 out of 200 个可能的)
测试集: 未测杂交组合 (其余 100 个 F1)

评估: 预测值与实际值之间的 Pearson 相关系数
```

典型结果：
- 玉米产量：r = 0.4-0.7
- 水稻产量：r = 0.3-0.6
- 油菜产量：r = 0.3-0.5
- 如果 r < 0.2 → 预测模型没有实际意义

## 杂交预测 vs 自交系直接选择：何时杂交预测值得投入？

杂交预测的成本（杂交制种 + 田间评价）远高于自交系选择。需要判断是否值得：

**杂交预测值得投入的条件**：
1. **杂种优势显著**：目标性状的杂种优势 > 15%（玉米产量、油菜产量通常符合，水稻较小）
2. **非加性遗传方差占比高**：SCA 方差/总遗传方差 > 0.3
3. **经济价值高**：杂交种的商业价值 >> 自交系（如玉米杂交种售价 5-10x）
4. **亲本数量大**：需要预测的组合 >> 能够实验的组合 (如 200 亲本 → 19900 个可能的组合 → 只能测 <5%)

**不值得做杂交预测的场景**：
- 自花授粉作物（如小麦、大豆）的自交系直接等同于品种
- 杂种优势 < 10%（如部分水稻性状）
- 亲本数量小 → 可以直接实验所有组合

## 作物特异性杂交育种策略

### 玉米 — 杂交育种的黄金标准

- **已建立的杂优群**：Stiff Stalk vs Non-Stiff Stalk（使用超过 50 年）
- **双列杂交规模**：通常 10-20 个测验种 × 100-500 个待测验系
- **预测准确性**：产量 r = 0.5-0.7（在所有作物中最高）
- **特殊考虑**：父本和母本的 GCA 方差通常不同 → 需要允许异方差模型

### 水稻 — 杂交稻的特殊性

- **两系/三系杂交稻**：利用雄性不育系统，杂交制种效率高
- **预测准确性**：产量 r = 0.3-0.5（低于玉米，因为杂种优势较小）
- **籼粳亚群间杂种**：籼粳交的杂种优势通常大于籼籼交和粳粳交
- **特殊考虑**：杂交稻主要在中国和东南亚种植，美洲/非洲以常规稻为主 → 杂交预测在这些地区的需求不同

### 小麦 — 杂交小麦的挑战

- **自花授粉**：天然异交率 <1%，制种困难导致杂交种子成本极高
- **杂种优势**：产量杂种优势通常 5-15%，低于玉米的 15-30%
- **杂交制种**：需要化学杀雄剂 (CHA)，增加了成本和复杂性
- **预测现状**：杂交预测在小麦中研究较少（因为杂交小麦本身比例低），大多数基因组预测用于自交系选择
- **建议**：先评估是否有杂交小麦的商业需求 → 如果有，用 small pilot factorial 设计测试 GCA/SCA 方差组成

### 油菜 — 杂种优势利用的中间案例

- **杂种优势强**：产量杂种优势 20-30%
- **恢复系-保持系系统**：类似水稻的雄性不育-恢复系统
- **挑战**：油菜是异源四倍体（AACC 基因组）→ 遗传模型需要考虑亚基因组效应

## 多性状杂交选择指标

杂交育种通常同时关注多个性状。单个性状最优的杂交组合可能在另一些性状上表现差。

### 常用选择指数

- **Smith-Hazel Index**：基于经济权重的多性状选择指数
- **Restricted Index**：约束某些性状不变，最大化其他性状
- **Desired Gain Index**：设定每个性状的目标遗传增益，选择满足所有目标的组合

### 实施示例

```
Smith-Hazel Index = w1*EBV_yield + w2*EBV_disease + w3*EBV_quality

其中 w1, w2, w3 是经济权重
```

**实际权重建议（玉米）**：
- 产量：0.6
- 抗倒伏：0.2
- 籽粒含水量（收获期）：0.15
- 品质性状（蛋白含量等）：0.05

**注意**：权重在不同育种项目中不同！没有通用的最佳权重。

## 最优配偶选择 (Optimal Mate Selection)

### 目标

1. 最大化遗传增益（后代平均表现）
2. 控制近交系数（长期遗传多样性）
3. 满足育种约束（如不能自交，偏好特定亲本组合）

### OCS (Optimal Contribution Selection) 方法

数学问题：选择一组亲本贡献比例 w_i，使得：
- 最大化 Σ(w_i * EBV_i)
- 约束：ΣΣ(w_i * A_ij * w_j) < C (C = 目标近交率)

### 育种约束

- 亲本不能自交
- 某些亲本不能互相杂交（如已知的不亲和性组合）
- 每个亲本最多参与 n 个杂交
- 总杂交组合数有限（育种规模约束）

## 常见陷阱

### 数据质量问题

- **表型数据缺失严重**：某些杂交组合只有一年数据 → 评估其 GCA 不稳定
- **试验设计不平衡**：不是所有亲本都一样多杂交组合 → 某些亲本的 GCA 估计不准确

### 群体结构混淆

- 如果所有"A 组"亲本都来自一个杂优群，"B 组"都来自另一个 → GCA 效应和群体效应混淆
- **检查**：PCA 中 A 组和 B 组是否在两个不同的 cluster

### GCA 高估与小测验种集问题

如果只用 2-3 个测验种评估大量候选亲本的 GCA：
- GCA 估计的标准误差大，排序不可靠
- 特别是候选亲本数量 > 50 时

**建议测验种数量**：
- 作物遗传变异小 → ≥ 3 个测验种
- 作物遗传变异大（如玉米） → ≥ 5 个测验种
- 测验种需要覆盖主要的杂优群

### 杂优群误分类

将属于同一杂优群的亲本错误放入不同杂优群 → 在"组间"杂交中丢失最佳组合 → SCA 被高估（因为群内杂交表现差，使得群间杂交的 SCA 看起来更大）。

**检测方法**：
- 比较分子标记划分和配合力划分的一致性
- 如果不一致 > 30% → 杂优群划分有问题

### 预测准确性的虚假膨胀

- **不要**在同一个杂交组合上同时训练和测试（数据泄露）
- **正确的交叉验证**：按杂交组合（而非亲本）做交叉验证
- **最严格的测试**：留出一个亲本的所有杂交组合（预测新亲本的杂交表现）

### 时间因素

植物育种中预测模型的主要问题：历史数据训练的模型在新一代材料上不准确。
- 原因是：基因频率、连锁不平衡、环境都在变化
- **应对**：使用"前向预测"（forward prediction）验证 → 在较早年份训练，在最新年份测试
