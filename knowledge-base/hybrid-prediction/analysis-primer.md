# 杂交预测结果解读

## 一句话解释

杂交预测：利用亲本信息（基因型/系谱/配合力）预测杂交组合的表现，筛选最优组合。

## 能回答什么

- 哪些亲本配组预期表现最好？
- GCA（一般配合力）和 SCA（特殊配合力）各贡献多少？
- 杂优群如何划分？

## 不能回答什么

- 预测在所有环境中都准确吗？（取决于训练环境范围）
- SCA 小的组合一定比 SCA 大的差吗？（取决于 GCA 水平）
- 杂优群划分是永久的吗？（随种质交流而变化）

## 典型输出

| 文件 | 含义 |
|------|------|
| `gca_sca_estimates.csv` | GCA/SCA 效应估计 |
| `predicted_hybrids.csv` | 预测的杂交组合排名 |
| `heterotic_groups.csv` | 杂优群划分结果 |
| `bakers_ratio.csv` | Baker's ratio 及遗传方差分量 |

## 常见结果模式

### Baker's ratio > 0.8 (GCA 主导)
水稻、小麦等自交作物常见。选择 GCA 高的亲本即可。SCA 不重要。

### Baker's ratio < 0.5 (SCA 主导)
玉米、油菜常见。特定组合效应大，必须测试很多组合。

### 分子标记 vs 配合力划分的杂优群不一致
应该结合两种方法：标记提供客观框架，配合力提供育种价值验证。
