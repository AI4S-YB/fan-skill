# 表型数据分析结果解读

## 一句话解释

表型分析：用统计模型从试验数据中提取遗传效应（育种值/BLUP），剔除环境噪声。

## 能回答什么

- 各基因型的 BLUP 值（最佳无偏预测）是多少？
- 性状的遗传力有多高？（决定 GWAS/GS 能否成功）
- 哪些基因型显著优于对照（多重比较）？

## 不能回答什么

- 为什么这个基因型表现好？（表型分析不解释机制）
- BLUP 值在不同环境中能直接比较吗？（需要多环境联合分析）
- 遗传力估计是"真实值"吗？（取决于试验设计和环境代表性）

## 典型输出

| 文件 | 含义 |
|------|------|
| `blup_values.csv` | 各基因型的 BLUP/BLUE 值 |
| `heritability.csv` | 遗传力估计 |
| `descriptive_stats.csv` | 均值/方差/CV/分布 |
| `anova_results.csv` | 方差分析表 |

## 常见结果模式

### 遗传力很低 (<0.2)
增加重复数或环境数。多环境联合分析可提高遗传力估计精度。低遗传力性状建议用 GS 而非 GWAS。

### CV > 30%
检查数据录入错误、试验设计问题、是否存在异常小区。田间试验中 CV=15-25% 常见。

### BLUP 和 BLUE 排名不同
BLUP 对极端值有 shrinkage（向均值收缩）→ 小样本/少重复的基因型 BLUP 被"拉向"均值更多。这是预期行为。
