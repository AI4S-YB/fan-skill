# 全基因组复制分析结果解读

## 一句话解释

全基因组复制分析：检测基因组中的多倍化事件，研究WGD历史及其对基因组进化的影响。

## 能回答什么

- 基因组经历过WGD吗？
- WGD发生在什么时候？
- WGD是共享的还是物种特异的？
- WGD后有多少基因被保留？
- 哪些类型的基因更容易保留？

## 不能回答什么

- WGD的具体原因？（环境压力、杂交等）
- WGD的适应性意义？（需要更多研究）
- 精确的WGD时间？（只有估计范围）

## 典型输出

| 文件 | 含义 |
|------|------|
| `ks_distribution.pdf` | KS分布图 |
| `collinearity.txt` | 共线性区块 |
| `dotplot.pdf` | 共线性点图 |
| `wgd_events.txt` | WGD事件列表 |
| `retained_genes.txt` | WGD保留基因 |
| `ancestral_karyotype.txt` | 祖先核型推断 |

## 常见结果模式

### KS分布单峰

一个清晰的KS峰通常代表一次WGD事件。峰的位置反映WGD的年龄。

### KS分布多峰

多个峰可能代表多次WGD事件，或不同类型的复制事件（WGD vs 串联复制）。

### 点图平行对角线

共线性点图中基因组自身的平行对角线是WGD的典型信号。

### 共线性深度分布

基因组内2:2或4:4的共线性模式分别暗示一次或两次WGD。

## 植物特异性结果解读

### 古多倍体

一些"二倍体"植物实际是古多倍体，KS分布可能有古老峰，大量基因有多个拷贝。

### 共享WGD

许多植物类群共享WGD事件，如十字花科的At-α、禾本科的ρ-WGD。

### 基因保留偏好

WGD后转录因子、信号分子等剂量敏感基因保留率较高；而代谢酶保留率较低。

## 结果报告建议

1. 报告检测到的WGD事件数量
2. 展示KS分布图并标注峰位置
3. 展示共线性分析结果
4. 估计WGD年龄并说明校准方法
5. 分析基因保留模式
6. 讨论WGD对基因组进化的影响
7. 与其他近缘物种比较WGD历史
