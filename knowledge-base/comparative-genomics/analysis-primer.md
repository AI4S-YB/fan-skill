# 比较基因组学分析结果解读

## 一句话解释

比较基因组学：通过比较不同物种的基因组，揭示基因组进化历史和功能演化。

## 能回答什么

- 物种之间的亲缘关系是什么？
- 哪些基因是保守的？哪些是物种特异的？
- 基因组经历过哪些WGD事件？
- 染色体结构发生过哪些重排？
- 哪些基因家族发生了扩张/收缩？

## 不能回答什么

- 基因的具体功能？（需要实验验证）
- 表型差异的遗传基础？（需要关联分析）
- 古老基因组的精确序列？（只能推断）

## 典型输出

| 文件 | 含义 |
|------|------|
| `Orthogroups.tsv` | 同源基因分组 |
| `Species_Tree.svg` | 物种系统发育树 |
| `collinearity.txt` | 共线性区块 |
| `ks_distribution.pdf` | KS分布图 |
| `circos.png` | 基因组比较环形图 |
| `cafe_results.txt` | 基因家族变化统计 |

## 常见结果模式

### 同源基因组

Orthogroups包含多个物种的直系同源基因，单拷贝直系同源组适合构建物种树。

### 共线性区块

共线性区块显示基因组间的保守区域，区块方向反映染色体重排历史。

### KS分布峰

KS分布中的峰代表WGD事件，峰的位置反映WGD的年龄。

### 基因家族扩张

特定基因家族在某些谱系扩张，可能与适应性进化相关。

## 植物特异性结果解读

### 多倍化历史

植物基因组常有多次WGD，KS分布可能出现多个峰。需要区分共享WGD和独立WGD。

### 染色体数目变化

植物染色体数目变化频繁，可能通过融合、分裂、多倍化等方式。

### 基因组大小差异

植物基因组大小差异主要源于重复序列，与基因数量相关性不高。

## 结果报告建议

1. 报告比较的物种数量和基因组质量
2. 展示物种系统发育树
3. 统计同源基因分布
4. 展示共线性分析结果
5. 分析WGD历史
6. 讨论基因家族进化
7. 总结基因组进化事件时间线
