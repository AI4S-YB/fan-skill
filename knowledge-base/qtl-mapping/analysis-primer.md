# QTL 作图结果解读

## 一句话解释

QTL 作图：在双亲分离群体中，利用遗传连锁图谱定位控制数量性状的染色体区域。

## 能回答什么

- 控制性状的 QTL 在哪些染色体上？
- 每个 QTL 的效应有多大（LOD、% 变异解释）？
- QTL 在不同环境中是否稳定？

## 不能回答什么

- QTL 区间内的哪个基因是因果基因？（需要精细定位 + 功能验证）
- QTL 效应在其他群体中也存在吗？（取决于亲本等位基因是否在其他群体中分离）
- 所有 QTL 都被检测到了吗？（微效 QTL 可能因功效不足漏掉）

## 典型输出

| 文件 | 含义 |
|------|------|
| `qtl_scan_results.csv` | 全基因组 LOD 曲线 |
| `qtl_effects.csv` | QTL 效应大小和置信区间 |
| `lod_profile.pdf` | LOD 曲线图（每条染色体） |
| `qtl_summary.csv` | QTL 汇总表 |

## 常见结果模式

### 零个 QTL
< 100 个体的群体中正常。功效不足是最常见原因。增加群体大小或环境数。

### QTL 置信区间很宽 (>20 cM)
遗传图谱密度低（<200 标记）。增加标记密度可以缩小置信区间。

### 多环境 QTL 不完全重叠
QTL×E 互作。联合分析（而非逐环境比较）能更好地估计 QTL×E。
