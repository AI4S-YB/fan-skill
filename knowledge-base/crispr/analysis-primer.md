# CRISPR 设计结果解读

## 一句话解释

CRISPR 基因编辑设计：为目标基因找到最优的 sgRNA 靶序列，评估编辑效率和脱靶风险。

## 能回答什么

- 哪些 sgRNA 靶序列最适合（高效率 + 高特异性）？
- 目标位点在基因组中有多少潜在的脱靶位点？
- 编辑后预期产生什么类型的突变？

## 不能回答什么

- 编辑效率的精确值？（仅预测，实际效率受转化/再生影响）
- 植物中的脱靶率是否如预测？（需要全基因组测序验证）
- 表型一定会出现吗？（基因冗余可能掩盖表型）

## 典型输出

| 文件 | 含义 |
|------|------|
| `sgRNA_candidates.csv` | 候选 sgRNA 序列及评分 |
| `offtarget_analysis.csv` | 潜在脱靶位点列表 |
| `primers.csv` | 基因分型验证用引物 |

## 常见结果模式

### 找不到合适 PAM 位点
用 Cas9-NG (PAM: NG) 或 Cas12a (PAM: TTTV) 扩展选择范围。

### 多倍体中脱靶位点过多
按亚基因组拆分搜索。若一个 sgRNA 在 2+ 亚基因组中有完美匹配 → 会同时编辑。
