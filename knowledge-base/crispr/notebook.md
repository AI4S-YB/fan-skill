# 植物 CRISPR/Cas9 设计：分析笔记

## 在开始之前

植物基因编辑的核心挑战来自三个方面：基因组复杂性（多倍体、重复序列）、转化效率（不同物种差异巨大）、以及同源基因干扰。先问自己：

### 你想达成什么编辑目标？

- **基因敲除 (Knockout/KO)**：产生移码突变 → 设计 sgRNA 靶向早期外显子，PAM 后产生 DSB，NHEJ 修复引入 indel
- **基因敲入 (Knock-in/KI)**：插入外源序列 → 需要 HDR 供体模板 + 同源臂设计
- **碱基编辑 (Base Editing)**：C-to-T 或 A-to-G 单碱基替换 → 需要目标碱基位于编辑窗口内（通常为 protospacer position 4-8）
- **Prime Editing**：精确的短片段插入/删除/替换 → 需要 pegRNA（prime editing guide RNA）+ nicking sgRNA
- **多重编辑 (Multiplex)**：同时编辑多个基因 → 多个 sgRNA 共表达 + 可能需要 Cas12a（更简单的 crRNA array）

### 选择正确的编辑工具：Cas9、Cas12 还是碱基编辑器？

| 工具 | 机制 | PAM | 适合的编辑目标 | 不适合 |
|------|------|-----|--------------|--------|
| SpCas9 | DSB → NHEJ indel | NGG | 基因敲除（KO） | 精确碱基替换、大片段删除 |
| Cas12a (Cpf1) | 交错 DSB、自主处理 crRNA | TTTV | 多重编辑（可用单一 crRNA array）；AT-rich 基因 | GC-rich 区域、NGG PAM 附近无合适靶位点 |
| Base Editor (ABE/CBE) | 脱氨酶介导的单碱基替换（无 DSB）| NGG (SpCas9 融合) | 点突变创建（C→T 或 A→G 替换）| 需要 indel 的敲除；PAM 位置要求严格 |
| Prime Editor | 逆转录酶 + nCas9 → 模板引导的精确编辑 | NGG (nSpCas9) | 精确短插入/删除/替换 (1-50 bp) | 大片段插入（>50bp 效率极低）|
| Cas9-NG | 扩展 PAM | NG | GC 含量高或 NGG 稀缺的区域 | 特异性略低于 SpCas9 |
| xCas9 | 最宽 PAM | NG/GAA/GAT | 靶位点 PAM 选择极受限时 | 效率在某些 PAM 下低于 SpCas9 |

**植物中工具选择的决策树**：

1. 目标是简单 KO → SpCas9（效率最高）
2. 需要同时编辑 3+ 个基因 → Cas12a（crRNA array 设计简单）
3. 目标是创建特定点突变（模拟自然等位基因）→ Base Editor 或 Prime Editor
4. 目标是精确插入小标签（如 HA/FLAG）→ Prime Editor（避免 HDR 的低效率）
5. 靶位点 GC 含量 > 65% 且 NGG 稀少 → Cas9-NG 或 xCas9

### 你的物种有什么资源？

- **主要作物**（水稻、玉米、拟南芥、大豆、番茄、小麦）→ CRISPOR 有预建参考基因组
- **非模式物种**→ CRISPR-P 允许上传自定义基因组，或使用 CHOPCHOP 命令行
- **多倍体**（小麦、棉花、油菜）→ 需要考虑同源基因共享 sgRNA 靶序列（同时编辑所有亚基因组拷贝）

### 你用什么 Cas 变体？

| Cas 变体 | PAM | 特点 | 植物应用 |
|----------|-----|------|---------|
| SpCas9 | NGG | 最常用 | 几乎所有植物物种 |
| Cas9-NG | NG | 放宽的 PAM | GC 含量高的植物基因组 |
| Cas12a (Cpf1) | TTTV | T-rich PAM, 交错切割 | AT-rich 区域 |
| Cas9-VQR | NGAN | 替代 PAM | 特定靶序列 |
| xCas9 | NG/GAA/GAT | 最宽 PAM | 靶序列 PAM 选择受限时 |

## 选择你的 sgRNA 设计工具

### CRISPOR — 主要作物首选

预建了主要植物物种的参考基因组，Web 界面易用。

**适合**：水稻、玉米、拟南芥、大豆、番茄、小麦
**优势**：综合评分（MIT specificity score + Doench efficiency score + 脱靶预测）

### CRISPR-P — 非模式植物

支持上传自定义基因组 FASTA。

**适合**：没有在 CRISPOR 中列出的植物物种
**优势**：支持自定义基因组、多倍体分析、基因家族考虑

### CHOPCHOP — 本地运行

命令行版可在本地/HPC 上运行。

**适合**：高通量设计（数百个基因）、自动化管道、自有基因组

## sgRNA 设计原则

### 最优 sgRNA 的特征

1. **位置**：CDS 前 1/3 处（产生早期的移码终止）
2. **特异性**：全基因组中独特的靶序列（避免同源基因同时被编辑）
3. **效率**：GC%=40-60%, 避免 4+ T 连续（Pol III 终止信号）
4. **PAM**：NGG（SpCas9）优先，检查是否有合适的 PAM 在基因组上下文中
5. **二级结构**：sgRNA 不应形成强发夹结构

### 启动子选择：U6 vs U3

在植物中，sgRNA 通常由 Pol III 型启动子驱动表达：

| 启动子 | 物种来源 | 转录起始位点要求 | 使用建议 |
|--------|---------|-----------------|---------|
| OsU6 | 水稻 | G 起始（转录起始的第一个碱基必须是 G）| 单子叶植物首选 |
| OsU3 | 水稻 | A 起始 | 单子叶植物备选（不能用于 G 起始的设计）|
| AtU6 | 拟南芥 | G 起始 | 双子叶植物首选 |
| TaU6 | 小麦 | G 起始 | 小麦等麦类作物 |

**重要规则**：
- 如果 sgRNA 靶序列以 G 开头 → 使用 U6 启动子
- 如果 sgRNA 靶序列以 A 开头 → 使用 U3 启动子
- 如果 sgRNA 靶序列以其他碱基开头 → 在 sgRNA 5' 端额外添加一个 G（使用 U6）→ 这会在 sgRNA 和目标序列间引入一个 G-T 错配，通常可容忍

**多倍体额外考虑**：不同亚基因组的 U6/U3 启动子可能活性不同 → 如果在一个亚基因组中有较高的编辑效率而另一个较低，尝试使用该亚基因组来源的启动子。

### 植物特异性考虑

- **靶向所有同源拷贝** vs **靶向特定亚基因组拷贝**：需要在 sgRNA 选择时明确
- 如果目标是在多倍体中敲除所有亚基因组拷贝，选择保守区域的 sgRNA
- 如果要研究亚基因组特异性功能，在外显子序列不同的区域设计 sgRNA
- 小麦中 3 个亚基因组（A、B、D）→ 通常需要 1 个保守 sgRNA 或 3 个特异性 sgRNA

## 脱靶分析

### Cas-OFFinder

全基因组脱靶搜索工具，Cas-OFFinder 可以搜索最多 3 个错配 + 1 个 DNA/RNA bulge 的脱靶位点。

**注意**：
- 植物基因组中重复序列多 → 某些 sgRNA 可能有数百个脱靶位点 → 应该排除
- 启动子区域的脱靶可能影响其他基因的表达
- 基因间区域的脱靶通常无害

### 大而重复的植物基因组中的脱靶挑战

植物基因组中的重复序列（特别是转座子衍生的重复）使脱靶预测特别困难：

**典型重复比例**：
- 玉米：85% 重复 → sgRNA 平均有 50+ 潜在脱靶位点
- 小麦：80% 重复 → 六倍体进一步增加了复杂性
- 水稻：45% 重复 → 相对可控

**应对策略**：
1. **提高特异性门槛**：植物中 CFD score > 90 为高特异性（vs 通用的 >80）
2. **使用配对 sgRNA (paired nickases)**：两个 Cas9 nickase 靶向相邻位点 → 理论上脱靶率降低 50-1500 倍
3. **截短 sgRNA (tru-gRNA)**：sgRNA 长度从 20nt 缩短到 17-18nt → 减少脱靶活性，但不降低 on-target 效率
4. **避开重复区域**：如果靶基因包含在重复序列中，优先选择位于独特序列中的 exon

### 特异性评分

- **CFD (Cutting Frequency Determination)** score：评估脱靶位点被切割的可能性
- **MIT specificity score**：CRISPOR 中使用的综合评分
- 一般标准：CFD score > 80 为高特异性
- **植物建议**：多倍体中 CFD score > 90 更安全

## 编辑效率预测

### DeepSpCas9

基于深度学习预测 SpCas9 在特定 sgRNA 靶序列上的编辑效率。

输入：30bp 序列（4bp upstream + 20bp protospacer + 3bp PAM + 3bp downstream）
输出：预测的 indel 频率

### 影响编辑效率的因素

1. **染色质可及性**：开放染色质区域的 sgRNA 效率更高（植物的 DNase-seq 或 ATAC-seq 数据可以辅助判断）
2. **GC 含量**：40-60% 最优
3. **PAM 邻近序列**：位置 17-20（靠近 PAM 的 4bp）对效率影响最大
4. **sgRNA 的二级结构**：折叠后正确的 scaffold 结构对于 Cas9 loading 至关重要

## HDR vs NHEJ：修复通路选择

### NHEJ (Non-Homologous End Joining)

植物中 DSB 修复的主要通路（>90% 的修复事件通过 NHEJ）。

**修复结果**：
- 小 indel（通常 1-10 bp）→ 移码突变 → 基因敲除
- 可预测性：低（不能精确控制 indel 大小和序列）

**适合**：基因敲除 — 目标仅仅是破坏蛋白功能

### HDR (Homology-Directed Repair)

需要供体模板，精确修复。

**修复结果**：
- 精确的插入/替换/修复
- 需要同源臂和供体模板

**植物中 HDR 的低效率问题**：
- 植物中 HDR 效率通常 < 0.1-5%（vs NHEJ 的 10-80%）
- HDR 主要在 S/G2 期活跃 → 植物体细胞大部分在 G1 期

### 何时选择 HDR over NHEJ

| 编辑目标 | 推荐通路 | 理由 |
|---------|---------|------|
| 创建 KO 突变 | NHEJ | 简单、高效、不需要供体模板 |
| 精确碱基替换 | Base Editor > HDR | 碱基编辑器效率远高于 HDR（10-50% vs <5%） |
| 插入短标签 (<50bp) | Prime Editor > HDR | Prime editing 在植物中效率优于 HDR |
| 插入大片段 (>100bp) | HDR（唯一选择） | 目前只有 HDR 能实现大片段插入 |
| 创建 reporter line（GFP 融合）| HDR | 需要精确的 in-frame 融合 |

### 提升植物 HDR 效率的策略

1. **同时抑制 NHEJ**：使用 NHEJ 抑制剂（如 Scr7）或 Ku70/Ku80 突变体背景
2. **增加供体模板量**：通过 geminivirus replicon 系统在植物体内大量扩增供体
3. **细胞周期同步化**：将植物细胞同步到 S/G2 期
4. **使用 Cas9 融合蛋白**：将 Cas9 与 DNA 修复蛋白（如 CtIP、hGemini）融合 → 局部促进 HDR

## HDR 供体模板设计

### Knock-in 策略

植物中 HDR（Homology-Directed Repair）效率远低于 NHEJ。策略包括：

1. **同时抑制 NHEJ**：使用 NHEJ 抑制剂（如 Scr7）或 Ku70/Ku80 突变体背景
2. **增加供体模板量**：通过 geminivirus replicon 系统在植物体内大量扩增供体
3. **同源臂长度**：通常 400-800bp
4. **共表达 Cas9 + sgRNA + 供体**：使用双 T-DNA 或 polycistronic 系统

### 供体模板设计要点

- 左同源臂 + 插入序列 + 右同源臂（全长 800-2000bp）
- 供体 PAM 应该被突变掉（silent mutation）以防止 Cas9 切割供体
- 插入序列 < 2kb（HDR 效率随插入长度指数下降）
- 供体 DNA 可以通过 particle bombardment 或 Agrobacterium 递送

## 递送方法选择

### 三种主要递送方法的比较

| 方法 | 原理 | 优点 | 缺点 | 适用物种 |
|------|------|------|------|---------|
| **农杆菌 (Agrobacterium)** | T-DNA 转移 Cas9 和 sgRNA 到植物细胞 | 高效、低成本、适合大规模 | 宿主范围限制（单子叶部分受限）；T-DNA 整合到基因组中 | 水稻、番茄、拟南芥、大豆 |
| **基因枪 (Particle bombardment)** | 金/钨颗粒携带质粒 DNA 穿透细胞壁 | 无宿主范围限制；可递送大片段 DNA | 效率较低；组织损伤大；拷贝数不可控 | 小麦、玉米（未成熟胚）、棉花 |
| **原生质体 (Protoplast)** | 去除细胞壁后 PEG/电穿孔转染 | 瞬时表达、快速验证 sgRNA 效率；DNA-free（RNP 转染） | 无法直接产生转基因植株（除非能再生） | 所有植物（但再生需要物种特异性方案）|

### 物种特异性递送建议

**水稻**：
- 首选农杆菌（遗传转化成熟高效）→ 50-90% 转化效率
- 原生质体用于 sgRNA 快速验证（一周内看到编辑结果）

**玉米**：
- 农杆菌介导的未成熟胚转化 → 5-20% 效率
- 对于困难基因型 → 基因枪 + 选择标记

**小麦**：
- 基因枪是主流方法（农杆菌转化在小麦中效率极低 1-3%）→ 1-10%
- 六倍体基因组大 → 需要更高剂量的 Cas9 和 sgRNA

**大豆**：
- 农杆菌（子叶节法）→ 1-5% 效率
- 大豆转化是主要转化物种中最低效的 → 降低期望，设计 4-6 条 sgRNA 增加成功概率

**拟南芥**：
- Floral dip（花浸沾）转化 → 最简单、最高效
- 无需组织培养 → 直接获得转化种子

## 基因分型验证

### 编辑检测方法

1. **T7E1/Surveyor assay**：酶切错配检测，快速但半定量
2. **PCR + Sanger 测序**：金标准，可以确定确切的 indel 序列
3. **ICE (Inference of CRISPR Edits)** 分析：Sanger 测序色谱图分解，估计编辑效率和基因型比例
4. **扩增子 NGS**：多重编辑时使用，高灵敏度检测低频率编辑事件

### 植物特异性验证

- T0 代通常是嵌合体（多种基因型共存于同一植株）
- 需要 T1 或 T2 代才能获得纯合编辑植株
- 分离"转基因"（Cas9/sgRNA）和"编辑"（突变）：在 T2 代可以筛选 transgene-free 编辑事件
- 体细胞编辑事件不遗传到下一代 → T0 的基因分型不能完全预测 T1 的结果

## 常见陷阱

### 多倍体的 sgRNA 特异性

- 在一个亚基因组中"独特"的靶序列可能与另一个亚基因组中的序列只有 1-2 个错配 → 仍可能被切割
- **检查**：将 sgRNA 比对到所有亚基因组，查看错配分布
- **策略**：如果不想编辑所有拷贝，选择 2+ 错配位于 PAM 近端（seed region）的 sgRNA

### 基因冗余

- 同一基因家族的成员可能导致"敲除表型缺失"（另一个家族成员补偿了功能）
- **策略**：先检查基因家族（OrthoFinder），如果是大家族，考虑多重敲除多个成员

### T0 嵌合体：最常见但被低估的问题

T0 代植物不是单细胞的克隆后代 → 体细胞中不同细胞可能携带不同的编辑事件：

- 一株 T0 植物的不同叶片可能含不同的突变
- 一片叶子的不同部分可能含不同突变
- 某个编辑事件可能不存在于生殖细胞谱系中 → 无法遗传

**缓解策略**：
- 使用卵细胞特异性启动子（如 EC1.2）驱动 Cas9 表达 → 编辑偏向生殖细胞
- 使用 RPS5A 启动子（组成型高表达）→ 增加体细胞编辑，间接增加生殖细胞编辑概率
- 在 T0 代检测多个叶片/分蘖 → 如果仅某个分蘖有 edit，标记该分蘖的种子
- T1 代必须做基因分型 → T0 嵌合体可能在 T1 表现出不同的等位基因

### 多倍体中的脱靶风险放大

六倍体小麦中，一个 sgRNA 的设计可能：
- 主靶：A 亚基因组 1 个完全匹配位点
- 脱靶 1：B 亚基因组 1 个 2-错配位点
- 脱靶 2：D 亚基因组 1 个 3-错配位点
- 脱靶 3-20：重复序列中的多个 3-4 错配位点

→ 实际脱靶位点数 = 二倍体的 3-5 倍

**应对**：多倍体中做脱靶分析时必须对所有亚基因组做全基因组搜索

### 再生瓶颈

编辑成功需要：转化 + 编辑 + 再生完整植株。许多植物物种的再生效率是主要瓶颈：

- 棉花：体细胞胚胎发生困难 → 从转化细胞到完整植株 < 5%
- 大豆：子叶节转化后不定芽诱导效率低 → < 10%
- 树木物种：组织培养再生系统未建立 → 可能需要 2-5 年才能获得植株

**应对策略**：
- 在开始编辑项目前，先确认该物种/基因型的再生 protocol 是否可用
- 使用发育调节因子（如 WUS2/BBM 或 GRF-GIF 嵌合蛋白）提高再生效率
- 如果再生是主要瓶颈 → 使用原生质体瞬时系统验证 sgRNA 效率，然后仅对最优 sgRNA 做稳定转化

### 转化效率差异

不同植物物种的转化效率天差地别：

| 物种 | 转化方法 | 典型效率 |
|------|---------|---------|
| 水稻 | Agrobacterium | 50-90% |
| 玉米 | Agrobacterium | 5-20% |
| 小麦 | Particle bombardment | 1-10% |
| 大豆 | Agrobacterium (cotyledonary node) | 1-5% |
| 棉花 | Agrobacterium | 5-15% |
| 番茄 | Agrobacterium | 10-50% |

如果转化效率 <5%，可能需要在 sgRNA 设计阶段选择 3-5 条 sgRNA 以增加成功概率。
