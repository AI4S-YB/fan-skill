# 基因家族分析结果解读

## 一句话解释

基因家族分析：鉴定基因组中具有共同祖先的基因集合，研究其进化历史和功能分化。

## 能回答什么

- 这个基因家族有多少成员？
- 成员之间的进化关系是什么？
- 哪些成员经历了正选择/负选择？
- 基因家族是如何扩张的（WGD/串联重复/片段复制）？
- 启动子区域有哪些调控元件？

## 不能回答什么

- 基因的具体功能是什么？（需要实验验证）
- 基因在哪些细胞类型中表达？（需要单细胞数据）
- 基因的调控网络是什么？（需要ChIP-seq等数据）

## 典型输出

| 文件 | 含义 |
|------|------|
| `gene_family_proteins.fasta` | 基因家族成员蛋白序列 |
| `aligned.fasta` | 多序列比对结果 |
| `*.treefile` | 系统发育树 |
| `meme.html` | Motif分析结果 |
| `*.collinearity` | 共线性分析结果 |
| `kaks_results.txt` | Ka/Ks计算结果 |
| `plantcare_results.txt` | 顺式元件分析结果 |

## 常见结果模式

### 系统发育树分组

基因家族成员聚类成几个亚族，每个亚族可能具有不同的功能。结合motif和基因结构分析验证分组。

### Ka/Ks < 1

负选择（纯化选择），说明基因功能保守，受到选择压力维持功能。大多数基因家族成员的Ka/Ks < 1。

### Ka/Ks > 1

正选择，说明基因功能正在快速进化。可能出现了新功能（neofunctionalization）或功能分化。

### 共线性区块

WGD产生的基因对通常位于共线性区块内，串联重复基因相邻排列。

### 顺式元件富集

某些元件类型（如ABRE、MBS）在特定亚族中富集，提示功能分化。

## 结果报告建议

1. 报告基因家族成员总数和鉴定方法
2. 展示系统发育树并标注支持率
3. 统计不同复制类型的基因对数量
4. 报告Ka/Ks分布和平均值
5. 总结motif和顺式元件分析结果
6. 讨论功能分化的可能机制
