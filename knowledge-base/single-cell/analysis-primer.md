# 单细胞RNA-seq分析结果解读

## 一句话解释

单细胞RNA-seq：在单细胞水平研究基因表达，揭示细胞异质性和细胞类型组成。

## 能回答什么

- 样品中有哪些细胞类型？
- 不同细胞类型的marker基因是什么？
- 细胞类型之间的发育关系是什么？
- 不同条件下细胞类型组成有何变化？
- 细胞状态转变的轨迹是什么？

## 不能回答什么

- 细胞的空间位置信息？（需要空间转录组）
- 细胞的表观遗传状态？（需要scATAC-seq）
- 蛋白质水平？（需要CITE-seq）
- 细胞功能的直接证据？（需要实验验证）

## 典型输出

| 文件 | 含义 |
|------|------|
| `filtered_cells.rds` | 过滤后的Seurat对象 |
| `umap_coordinates.tsv` | UMAP坐标 |
| `clusters.tsv` | 聚类结果 |
| `markers.tsv` | Marker基因列表 |
| `cell_annotations.tsv` | 细胞类型注释 |
| `trajectory.pdf` | 轨迹分析图 |

## 常见结果模式

### 清晰的细胞类型聚类

UMAP图上不同cluster明显分离，cluster内marker基因特异性高表达，说明细胞类型鉴定可靠。

### 连续的发育轨迹

拟时序轨迹显示细胞状态的连续变化，分支点代表命运决定，提示发育路径。

### 批次效应

不同批次的细胞在UMAP上分离而非按生物学类型聚类，需要进行批次校正。

### 细胞周期效应

细胞沿细胞周期方向分离，如果与生物学问题无关，需要回归掉。

## 植物特异性结果解读

### 组织特异性细胞类型

植物不同组织的细胞类型差异大，建议分组织解读。叶片、根、茎的细胞类型组成完全不同。

### 分化程度差异

植物组织中可能同时存在分生组织细胞（未分化）和成熟细胞（已分化），表达模式差异大。

### 环境响应

植物细胞对环境（光、温度、胁迫）响应强烈，可能影响细胞状态和基因表达。

## 结果报告建议

1. 报告质控指标和过滤标准
2. 展示UMAP/t-SNE聚类图
3. 列出每个cluster的top marker基因
4. 展示细胞类型注释结果
5. 如有轨迹分析，展示拟时序图和命运基因
6. 讨论细胞类型组成与生物学问题的关联
