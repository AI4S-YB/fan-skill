# 基因组注释结果解读

## 一句话解释

基因组注释：在基因组序列上预测蛋白质编码基因、RNA 基因和其他功能元件的位置和结构。

## 能回答什么

- 基因组有多少个蛋白质编码基因？
- 每个基因位于哪里？有几个外显子？
- 基因有什么功能（GO/KEGG/Pfam）？

## 不能回答什么

- 预测的基因一定在体内表达吗？（需要 RNA-seq 验证）
- 功能注释一定正确吗？（跨物种推断有不确定性）
- 基因数量异常少或多？（需评估注释完整性）

## 典型输出

| 文件 | 含义 |
|------|------|
| `genome.gff3` | 基因/外显子/CDS 坐标 |
| `proteins.faa` | 预测的蛋白质序列 |
| `functional_annotation.tsv` | GO/KEGG/Pfam 注释 |
| `busco_summary.txt` | 注释完整性评估 |

## 常见结果模式

### 基因数量异常多 (>2x 近缘种)
可能原因：重复序列未充分屏蔽、单倍型组装冗余、可变剪切被计为独立基因。

### BUSCO Complete < 70%
增加 RNA-seq 证据或调整 BRAKER3 参数。检查组装是否片段化。

### 基因数量异常少
BRAKER3 参数过严、RNA-seq 覆盖不足、组装片段化打断基因。
