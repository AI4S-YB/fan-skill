# 代谢组学结果解读

## 一句话解释

代谢组学：系统地检测和定量生物样品中的小分子代谢物。

## 能回答什么

- 实验条件下哪些代谢物发生了显著变化？
- 变化的代谢物属于哪些类别（黄酮、生物碱、有机酸等）？
- 哪些代谢通路受到调控？

## 不能回答什么

- 所有变化代谢物都能鉴定吗？（非靶向代谢组学中大量 features 无法鉴定）
- 代谢物变化一定对应表型变化吗？（相关性 ≠ 因果）
- 定量是绝对的吗？（非靶向通常是相对定量）

## 典型输出

| 文件 | 含义 |
|------|------|
| `feature_table.csv` | 所有检测到的代谢物特征及丰度 |
| `differential_metabolites.csv` | 显著差异代谢物 |
| `metabolite_annotations.csv` | MS/MS 或数据库匹配的鉴定结果 |
| `pathway_enrichment.csv` | 差异代谢物的通路富集 |

## 常见结果模式

### 大量 features 无法鉴定 (>50%)
植物次生代谢物大多未收录在公共数据库中。正常现象。报告鉴定率，对未鉴定但显著的 features 诚实标注。

### 处理组间无显著差异
检查基质效应是否严重。使用 QC 样本做信号校正。n=3 功效可能不足。
