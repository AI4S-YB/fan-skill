# 泛基因组结果解读

## 一句话解释

泛基因组：一个物种全部个体/品种的所有基因的集合，包含核心基因（所有个体共有）和可变基因（部分个体特有）。

## 能回答什么

- 核心基因组包含多少基因？
- 哪些基因存在/缺失变异 (PAV)？
- 可变基因富集在哪些功能类别？
- SV 是否与表型变异关联？

## 不能回答什么

- 私有基因一定是真实的吗？（部分可能是组装/注释假象）
- PAV 基因一定影响表型吗？（可能被功能冗余补偿）
- 图基因组比传统参考好在哪？（需要具体问题具体分析）

## 典型输出

| 文件 | 含义 |
|------|------|
| `core_variable_genes.csv` | 核心/可变/私有基因分类 |
| `pav_matrix.csv` | 基因存在/缺失矩阵 |
| `sv_catalog.vcf` | 结构变异目录 |
| `pangene_frequency.pdf` | 基因频率分布图 |

## 常见结果模式

### 核心基因组占比过低 (<30%)
检查：基因组数量是否太少？组装质量是否参差不齐？如果包含远缘品种/野生种，核心缩小是正常的。

### 私有基因过多 (>15%)
可能包含组装/注释假象。对私有基因做 BLAST 确认在其他基因组中确实不存在（而非组装 gaps）。

### 抗病基因 (NBS-LRR) PAV 特别高
完全正常。抗病基因家族在不同品种间存在/缺失变异极其丰富。
