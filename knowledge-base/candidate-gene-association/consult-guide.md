# 候选基因关联分析引导

## 这个分析解决什么问题

用户有一组预先选定的候选基因区域，想在这些区域内寻找与性状相关的序列变异（SNP 或单倍型）。典型表述：
- "我有 306 个耐盐候选基因，想做关联分析"
- "这几个 QTL 区间里哪些基因和表型相关"
- "已知功能基因的等位变异与性状的关系"
- "候选基因里有显著关联的 SNP 吗"
- "这些基因的单倍型有没有表型效应"

与全基因组 GWAS 的区别：
- 不扫描全基因组，只在候选区域内分析
- 检验次数少 → 多重检验校正策略不同 (FDR vs Bonferroni)
- 可以关注基因级单倍型而非单个 SNP
- 统计框架从"全基因组显著性"转向"候选集显著性"

## 你需要问什么

每次只问一个问题。根据用户已说的跳过已知信息。

1. **候选基因来源**: 文献？QTL 区间？功能注释？基因数量？
   - 来源决定置信度：功能验证过的基因 > 同源推断 > 纯生物信息预测
2. **基因型数据**: 候选区域内的 SNP 密度？VCF 格式？有多少缺失？
3. **群体**: 什么物种？样本量？亚群结构？自交还是异交？
   - 自交物种的单倍型可以直接从纯合位点推断，异交物种需要统计 phasing
4. **表型**: 什么性状？数值型还是等级型？几个重复/环境？
5. **分析粒度**: 想看单个 SNP 还是基因级单倍型？
   - 高密度标记 → 推荐基因级单倍型，降维减少检验次数
   - 低密度标记 → 每个 SNP 独立检验

## 什么时候推荐这个分析

- 用户明确有一组候选基因/区域
- 基因数量在几十到几百个（不是几个，也不是几千个）
- 不是全基因组扫描 (全基因组 → gwas)
- 不是双亲群体 QTL (双亲 → qtl-mapping)
- 候选基因来源有依据（即使只是"我感兴趣这几个"也可以，关键是用户知道自己在做候选基因分析而非探索性扫描）

## 植物特定考量

- **自交物种的单倍型优势**: 水稻、大豆、小麦等自交物种的个体高度纯合，单倍型可以直接从纯合 SNP 组合读取，不需要统计 phasing。这让基因级单倍型分析特别简单可靠。
- **候选基因选择来源**: 植物中最常见的候选基因来源包括 QTL 区间（最可靠）、BSA-seq 定位区间、同源基因推断（拟南芥/水稻功能基因在目标物种中的同源）、差异表达基因（RNA-seq）、和文献报道的功能基因家族成员。
- **稀有变异**: 植物育种群体中，一些功能等位变异可能频率很低（如抗病基因的稀有抗性等位基因）。候选基因分析中 MAF 阈值可以比 GWAS 更宽松（如 MAF > 0.01），因为检验次数少，假阳性风险可控。
- **多年生 vs 一年生**: 一年生作物（水稻、玉米、大豆）通常有成熟的自交系/品种群体做候选基因分析；多年生作物（果树、林木）的候选基因分析通常使用全同胞家系或无性系群体，单倍型推断策略不同。
- **关联验证路径**: 候选基因关联分析的阳性结果建议配套验证策略：近等基因系（NIL）、基因编辑（CRISPR）或至少在不同群体中重复验证。候选基因来自 QTL 定位时，可与原始 QTL 结果交叉验证。
