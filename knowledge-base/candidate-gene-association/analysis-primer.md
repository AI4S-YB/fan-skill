# 候选基因关联分析：结果解读

## 一句话解释

候选基因关联分析：在你预先选定的基因或基因组区域内，寻找与性状显著相关的序列变异（SNP 或单倍型）。

## 能回答什么

- 候选基因中哪些与性状显著相关？
- 每个显著基因的效应方向和效应量有多大？
- 基因的不同单倍型对表型值的影响（如单倍型 A 比 B 的耐盐性高 X%）
- 候选基因的排名：哪个最可能是因果基因？

## 不能回答什么

- 候选列表之外的基因是否相关？（分析范围限定在候选集内）
- 这个 SNP/基因就是因果位点吗？（需要功能验证，如基因编辑/过表达）
- 全基因组还有哪些区域可能相关？（这是 GWAS 的任务）
- 关联是因果关系还是 LD 拖带？（候选区域内的 LD 结构可能导致邻近基因被连带显著）

## 典型输出

| 文件 | 含义 |
|------|------|
| `gene_association.csv` | 每个候选基因的关联统计（检验方法、p 值、FDR 校正 p 值、效应量） |
| `haplotype_effects.pdf` | 各单倍型组间的表型箱线图或均值比较图 |
| `manhattan_plot_candidate.pdf` | 候选区域的曼哈顿图（x 轴是候选基因而非全基因组） |
| `top_candidates.csv` | 多维排序后的优先候选基因列表 |

## 常见结果模式

### 零个显著基因
不是失败。可能原因：
1. 这些候选基因确实与性状无关 → 候选集选择偏误（最常见）
2. 标记密度不够，区域内因果变异未被捕获
3. 样本量太小，功效不足以检测效应
4. 性状受多个微效基因控制，单个基因效应太小
→ 诚实报告。科学结论："在当前候选集中，未发现与性状显著相关的基因。"

### 多个显著基因，效应中等
常见结果。多维排序可以帮助区分优先级：
- p 值最小的不一定效应最大
- 效应量最大且 p 值显著的基因更可能是真实的

### 单个基因极度显著，其他不显著
可能是一个主效基因。这是个好消息，但需要确认：
- 该基因周围是否有其他基因也连带显著（LD 拖带）？
- 该基因的功能注释是否与性状生物学吻合？

### FDR 校正后全部不显著
说明原始 p 值不够强。可能：
- 候选集很大但效应都很弱
- 表型测量噪声大
→ 报告原始 p 值排序，明确标注"FDR 校正后不显著，需独立验证"
