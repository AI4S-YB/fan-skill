# 植物 GWAS：分析笔记

这份笔记帮助你像一个有经验的植物遗传学家一样思考和做决策。

## 在开始之前：理解你的实验材料

`inspect_data.sh` 产出了 data_profile.json。先不着急想方法，先理解你面对的是什么。

### 这是什么物种？

查看 `species_detected` 和 `breeding_system` 字段。如果物种已识别：
→ 查阅 `bio-plant-infra/references/species-cheatsheet.md` 了解该物种的特性

如果物种未识别：
→ 检查染色体命名模式：`chr1..chr10` 是玉米，`Chr01..Chr12` 是水稻，`1A..7D` 是小麦
→ 让用户确认物种，**不要猜测**

### 这个 SNP 密度能做什么？

看 `snp_density_quality` 字段：

- **very_low (<1000 SNP)**：这是候选基因标记集，不是全基因组。放弃 GWAS 思维，转向候选基因关联分析。可以用 GAPIT GLM + 已知功能注释。不要期望全基因组水平的发现——诚实地说明这点。

- **low (1000-10000)**：勉强能做 GWAS，但精细定位不可能。显著位点的置信区间会很宽（>1Mb）。用 GAPIT CMLM。Bonferroni 阈值用 0.05/nSNP 而非 5e-8。

- **medium (10K-100K)**：可以做规范的 GWAS。如果是自交物种，CMLM；如果是异交物种且 λ<1.05，可以考虑 FarmCPU。

- **high (>100K)**：全基因组扫描的标准密度。根据样本量和群体结构选择 FarmCPU 或 BLINK。

### 样本量意味着什么？

- **n < 100**：只能发现解释 >10% 表型变异的超大效应位点。0 个显著位点是正常结果，不代表分析失败。
- **n = 100-300**：中等功效。预期 0-5 个显著位点，取决于性状遗传结构。
- **n > 500**：功效足够检测中等效应位点。可以考虑 BLINK。
- **n > 2000**：功效充足，适合所有方法。

### 有多个环境吗？

植物育种最特别的一点——同一个基因型经常在多个环境中被评估。

- env_count = 1 → 标准单环境 GWAS
- env_count ≥ 2 → 考虑 MET-GWAS 策略

多环境数据有两种思路：
1. **Per-environment + 比较**（更直观）：分别跑 GWAS，找一致信号
2. **BLUP + GWAS**（更简洁）：先算跨环境 BLUP，再做一次 GWAS

哪种思路更好？如果环境差异主要来自年际/地点变异（不是处理差异），用 BLUP。如果不同环境有不同的处理（如正常 vs 干旱），建议 per-environment。

## 选择你的 GWAS 方法

这里没有唯一正确答案。每种方法有不同的假设：

### GAPIT CMLM — 最稳健的选择

用 K 矩阵建模样本间的亲缘关系，同时加 PCA 协变量控制群体结构。

**适合**：
- 近交物种（水稻、大豆、小麦）——K 矩阵在此类物种中不可或缺
- 小到中等样本量
- 任何标记密度
- 你有理由相信个体间存在复杂的亲缘关系

**不适合**：
- 没有

CMLM 是最"安全"的选择。如果你不确定该用什么方法，用 CMLM 不会出大错。

### FarmCPU — 平衡功效与假阳性

交替使用固定效应模型和随机效应模型。

**适合**：
- 群体分层明显（λ>1.05 或 PC1>10%）
- 中等以上标记密度（>1000 SNP）
- 你想控制假阳性但不牺牲太多功效

**不适合**：
- 极低密度标记（<1000 SNP）——多基因背景无法估计
- 极小的样本量（<100）

### BLINK — 最高功效

基于 LD 的分箱迭代方法。

**适合**：
- 大样本（>500）+ 高密度（>50K）
- 群体结构已由 PCA 或实验设计控制
- 你追求最大统计功效

**不适合**：
- 低密度标记（BLINK 假设了 SNP 间的 LD 关系）
- 强群体分层（需先用 FarmCPU）
- 自交物种小群体（K 矩阵缺失使假阳性升高）

### MET-GWAS — 多环境

**适合**：
- 数据跨 2 个以上环境
- 想区分稳定 QTL 和环境特异 QTL

## 常见陷阱

### λ 偏高怎么办？

先别急着加 PC。看一下 PCA 图——如果样本明显聚成几簇，那 λ 偏高是正常的群体分化信号，不是"问题"。

- 自交作物的 λ 通常比异交作物高，1.1-1.3 可以接受
- 如果 λ>2.0：严重分层。先按亚群分拆，各亚群分开分析
- 如果加了 K 矩阵后 λ 仍 >1.5：检查是否存在极端离群样本

### 零个显著位点

不要沮丧。最常见的原因是功效不足。报告中诚实说明：
- "当前标记密度（X SNPs）的统计功效不足以检测微效位点"
- "该性状可能由多个微效基因控制，需要更大样本量"
- 可以报告建议性阈值（1e-5）下的位点，但明确标注"未达显著性"

### 着丝粒区域的显著位点

如果 peak 落在着丝粒区域（通常表现为连续的多个显著 SNP），要特别小心：
- 着丝粒区域的重组率极低 → LD 异常高 → 假阳性风险大
- 需要参考基因组注释区分着丝粒/pericentromeric 区域
- 报告时标注"位于着丝粒附近，需独立验证"

### 候选基因注释

显著位点落在基因间区是最常见的情况。不要只报"基因间区"就结束：
1. 计算 LD 衰减范围 → 划定候选区间
2. 区间内的所有基因都可能是候选
3. 查表达数据库看这些基因在相关组织中是否表达
4. 查拟南芥/水稻同源基因的功能 → 提供跨物种推断
