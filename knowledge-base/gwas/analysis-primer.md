# GWAS 结果解读

## 一句话解释

全基因组关联分析 (GWAS)：在全基因组范围内寻找与你的性状相关的遗传位点。

## 能回答什么

- 哪些基因组区域可能控制这个性状？
- 这些区域的影响有多大（解释多少表型变异）？
- 不同染色体上是否有多个关联信号？

## 不能回答什么

- 这个位点就是因果基因吗？（需要功能验证）
- 为什么品种 A 的性状比品种 B 好？（GWAS 看群体内变异）
- 这个标记在不同环境中还能用吗？（需多环境验证）

## 典型输出

| 文件 | 含义 |
|------|------|
| `significant_snps.csv` | 显著关联的 SNP 列表 |
| `manhattan_plot.pdf` | 曼哈顿图：山峰越高，关联越强 |
| `qq_plot.pdf` | QQ 图：评估模型拟合质量 |

## 常见结果模式

### 零个显著位点
不是实验失败。可能原因：
1. 标记太少，功效不足（最常见）
2. 性状由多个微效基因控制
3. 表型测量误差大
→ 诚实报告。这是科学结果。

### λ (lambda) 值偏高 (>1.05)
存在群体分层。对自交作物 (水稻/大豆) 来说 1.1-1.3 可以接受。
如果 >2.0，需要按亚群分拆分析或加 K 矩阵。

### 最显著 SNP 在基因间区
完全正常。GWAS 的大部分信号在非编码区。需要：
1. 根据 LD 衰减范围划定候选区间
2. 列出区间内所有基因
3. 查这些基因的表达和功能
