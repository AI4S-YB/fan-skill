# 三维基因组分析结果解读

## 一句话解释

三维基因组分析：研究基因组在细胞核中的空间折叠方式，揭示染色质交互与基因调控的关系。

## 能回答什么

- 基因组的空间组织结构是什么？
- 哪些区域形成染色质Loop（增强子-启动子交互）？
- 基因组如何划分为TAD（拓扑关联域）？
- A/B区室的分布如何？
- 不同条件下三维结构有何变化？

## 不能回答什么

- 染色质交互的直接功能证据？（需要ChIP-seq等验证）
- 单细胞水平的结构差异？（需要single-cell Hi-C）
- 动态的染色质运动？（需要live-cell imaging）

## 典型输出

| 文件 | 含义 |
|------|------|
| `inter.hic` | .hic格式contact矩阵 |
| `matrix.cool` | .cool格式contact矩阵 |
| `loops.bedpe` | Loop位点列表 |
| `tads.bed` | TAD边界列表 |
| `compartments.bed` | A/B区室注释 |
| `differential.tsv` | 差异交互结果 |
| `contact_map.png` | Contact map可视化 |

## 常见结果模式

### Contact Map对角线特征

清晰的对角线表示良好的数据质量。对角线外的信号表示long-range交互。方块状结构表示TAD。

### Loop点

Contact map上对角线两侧的焦点状信号。通常对应增强子-启动子交互或CTCF介导的环状结构。

### TAD边界

TAD边界通常富含CTCF位点、启动子和活跃染色质标记。边界区域可能有重要的调控功能。

### A/B区室

A区室（正值）对应开放染色质、高基因密度、高表达区域。B区室（负值）对应异染色质、低基因密度区域。

## 植物特异性结果解读

### TAD边界可能不清晰

植物基因组的TAD边界通常不如动物基因组清晰。这与植物缺乏CTCF等绝缘子蛋白有关。

### 染色体疆域

植物细胞核中染色体形成明显的疆域（chromosome territories）。Contact map上可以看到清晰的染色体分界。

### 核仁组织区

植物基因组的rDNA区域通常位于核仁内，形成特殊的contact模式。

## 结果报告建议

1. 报告测序深度和比对质量指标
2. 展示典型染色体的contact map
3. 统计Loop和TAD数量
4. 分析Loop与基因表达的关系
5. 比较A/B区室与表观基因组数据
6. 讨论与基因调控的关联
