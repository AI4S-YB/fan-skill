# 植物群体遗传学：分析笔记

## 在开始之前

植物群体遗传学分析要回答的核心问题：
1. 这些样本之间有没有遗传亚群？
2. 有几个亚群？
3. 亚群之间差异多大？
4. 哪些样本属于哪个亚群？

## LD Pruning 为什么重要

PCA 和 ADMIXTURE 假设 SNP 是独立的。LD 会导致某些基因组区域的信号被过度放大。

- 高密度标记（>50K SNP）：必须 pruning，用默认参数（window=50, step=5, r2=0.2）
- 低密度标记（<10K SNP）：r² 阈值适当放宽（0.3-0.5），否则 prune 掉太多标记
- 如果 LD pruning 移除了所有 SNP（可能 < 100 剩余）：降低 r² 到 0.5 或跳过 pruning

## PCA 结果怎么看

### PC1 解释方差

- **> 30%**：极强的群体分化。可能存在亚种级别或不同育种群的分离。检查样本元数据（品种类型、地理来源）。
- **15-30%**：明显的群体结构。通常对应不同育种项目、生态型或地理区域。
- **5-15%**：适度结构。可能与育种历史或微弱的地理隔离有关。
- **< 5%**：群体基本均匀（对自交作物很少见）。

### 植物的"正常"范围

- 野生种质资源的 PC1 通常 > 20%（地理隔离强）
- 现代育种材料的 PC1 通常 5-15%（种质交流频繁）
- 如果 PC1 > 30%，优先分亚群再做下游分析

### 多倍体的特殊处理

- 异源多倍体（小麦、棉花、油菜）：做 per-subgenome PCA
- 同源多倍体：PCA 解释需谨慎，可能混合亚基因组+群体结构信号

## ADMIXTURE

### 选择最佳 K

不要只看 CV error 最低的 K。结合：
1. CV error 的肘点（elbow point）
2. 生物学合理性（已知有多少个品种/群体？）
3. 各 K 值的 bar plot — 是否存在从某个 K 开始出现"杂乱"的混合模式

### 植物中常见的模式

- 育种群体的 ADMIXTURE 通常呈现渐渗模式（introgressed pattern），而非离散簇
- 近交物种的个体可能在某些 K 下显示为多个祖先成分的混合，这不一定是真正的"混合祖先"而是 LD 的影响
- 野生群体更可能呈现离散的祖先成分

## Fst

植物比人类通常有更高的 Fst：
- 野生群体间 Fst 0.2-0.5 正常
- 育种群体间 Fst 0.05-0.15 常见
- 地理隔离群体间 Fst 可达 0.5+

## 系统发育树 (NJ)

对植物群体的 NJ 树：
- 近交物种的个体直接聚类（每个品种→明确的簇）
- 异交物种的树通常更"星形"（大量的内部分支）
- 如果树种出现与 PCA 不一致的拓扑结构，优先相信 PCA（NJ 对缺失数据和 rare variants 敏感）

## 把结果串起来

多个方法交叉验证：
1. PCA 的分群模式 → NJ 树的聚类模式 → ADMIXTURE 的祖先成分 → 三者一致 → 结论可靠
2. 如果不一致：通常是 ADMIXTURE 的 K 选择问题，或数据中存在 close relatives 扭曲了距离矩阵
