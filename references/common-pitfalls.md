# 常见分析陷阱

咨询过程中遇到以下情况时，向用户发出明确预警。每一项包括：用户可能说什么、为什么是问题、该怎么做。

---

## 实验设计类

### "我有 2 个重复，应该够了吧？"

**为什么是问题**：2 个生物学重复的方差估计极不稳定。差异表达工具（DESeq2/edgeR）虽然能跑，但：
- 假阳性率高（把噪声当成显著差异）
- 假阴性率也高（真正的差异基因检不出来）
- 不同工具之间的结果一致性差

**建议**：增加到 ≥3 个重复。如果实在无法增加，结果只能作为"候选列表"——所有显著基因都需要用其他方法（qPCR）独立验证。

### "我只测了处理组，没测对照组"

**为什么是问题**：没有对照组，无法区分"处理引起的表达变化"和"本来就有（或技术噪声引起）的表达水平"。

**建议**：
- 如果能补测对照组，这是最好的
- 如果不能：可以估计每个基因的绝对表达量，可以看"哪些基因表达量最高"，但不能说"哪些基因因处理而改变"
- 如果你的目的是看不同组织/发育阶段间的表达差异（如根vs叶），这就是有对照的

### "我的表型数据用的是高/中/低分类"

**为什么是问题**：GWAS 和 GS 的统计模型需要数值型表型。分类表型的信息量远低于原始测量值——"高"和"非常高"被归为同一类，损失了效应大小的估计能力。

**建议**：
- 尽可能用原始测量值（kg、cm、天等）
- 如果只有分类数据：对于 GWAS，可以用 case-control 设计；对于 GS，需要用阈值模型

### "我的样本来自不同年份和地点，混在一起分析了"

**为什么是问题**：环境差异（年际气候、土壤条件、管理方式）是植物实验中的主要变异来源。不建模环境效应会导致：
- 环境差异被误判为遗传效应（假阳性）
- 或环境差异掩盖真实的遗传效应（假阴性）

**建议**：将"环境"作为一个因子纳入分析模型。对于 GWAS，用多环境联合分析或每个环境单独分析后比较。对于 GS，用 G×E 模型。

## 数据质量类

### "我的标记缺失率比较高（>15%）"

**为什么是问题**：高缺失率会降低有效标记数、可能引入偏倚（如果缺失不是随机的）、导致部分样本因缺失过多被排除。

**建议**：
- 先做基因型填充（imputation），用 Beagle 或 IMPUTE2
- 如果缺失集中在某些样本：检查这些样本的 DNA 质量
- 如果缺失集中在某些染色体区域：可能是拷贝数变异，需要特别处理

### "我用的是几十年前的标记数据（如 RFLP/AFLP）"

**为什么是问题**：老式标记密度极低（通常 <500），且物理位置不确定。做 GWAS 只能定位到非常大的区间（>10Mb），很难找到候选基因。

**建议**：
- 如果能重新做基因分型（GBS 或 SNP 芯片），强烈建议
- 如果只能用老数据：降低期望——可能只能确认已知的 QTL 大致位置
- 可以考虑用这些标记做群体遗传结构分析，物理位置不那么关键

### "我的参考基因组是近缘种的，不是同一个种"

**为什么是问题**：近缘种的参考基因组可以用于 reads 比对，但比对率会降低，部分基因可能缺失或注释错误。

**建议**：
- 用于定量（如 RNA-seq counts）：可以用，但需要检查比对率（应该 >70%）
- 用于精细定位（如 GWAS 后的候选基因）：近缘种参考的价值有限
- 如果比对率 <60%：考虑做 de novo 组装

## 结果解释类

### "没有找到显著位点，是不是实验失败了？"

**为什么是问题**：不是。0 个显著位点可能因为：
1. 统计功效不足（最常见）— 标记太少、样本太小、效应太小
2. 该性状确实由许多微效基因控制（无主效 QTL）
3. 表型测量误差大，掩盖了遗传信号
4. 群体分层未被充分校正

**建议**：
- 诚实报告。"在当前样本量和标记密度下未检测到显著关联"是合理的科学结论
- 检查统计功效（用 GCTA 或 R 包做 power analysis）
- 如果多环境下都无显著位点：可能确实没有大效应 QTL
- 这不代表"数据没用"——这些数据仍然可以做基因组选择

### "最显著的 SNP 落在基因间区，怎么办？"

**为什么是问题**：完全正常！GWAS 显著位点落在基因间区是最常见的情况。这不一定是个问题，因为：
- 该 SNP 可能通过 LD 与远处的调控元件关联
- 基因间区可能含有增强子、启动子等调控元件
- 候选区间可能覆盖邻近的几个基因

**建议**：
1. 根据该物种的 LD 衰减范围划定候选区间（如水稻 ±200kb）
2. 列出区间内所有基因
3. 查这些基因在相关组织中是否表达
4. 查拟南芥/水稻同源基因的功能
5. 区间内的所有基因都是候选——不要只盯着离 SNP 最近的

### "差异表达的基因太多了（>30%），正常吗？"

**为什么是问题**：如果 >30% 的基因都显著差异表达了，最大的可能性不是"处理真的改变了这么多基因"，而是：
- 处理剂量太高（接近致死剂量，细胞在全面应激）
- 存在未校正的批次效应（如处理组和对照组在不同日期建库）
- 发育阶段的差异被误判为处理差异（处理组和对照组不在同一个发育阶段）

**建议**：
1. 看 PCA 图——PC1 是否完全按处理/对照分离？还是按建库日期分离？
2. 检查 top DEGs 是否具有生物学意义（如胁迫响应基因富集是合理的）
3. 如果是剂量问题：这仍然是有效的结果，报告时解释"在高剂量处理下观察到了全局转录重编程"

### "富集分析没有显著结果"

**为什么是问题**：可能是因为：
1. DEGs 太少（<30 个）→ 统计功效不足以检测通路富集
2. 该物种的通路注释不完整 → 很多基因无法映射到通路
3. 受影响的确实是多个不相关的通路 → 没有单一通路显著富集

**建议**：
- DEGs <30：不做富集分析，直接逐个查阅 top DEGs 的功能
- 注释不完整：尝试 Mercator 在线注释工具（适合非模式植物）
- 无显著通路：报告"差异表达基因分散在多个通路中，无单一通路显著富集"——这也是发现
