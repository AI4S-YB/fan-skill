# 面向生物学家的分析方法速查 -- 索引

每项分析的详细"白话说"解读已移至各自的知识条目中。此文件为索引。

## 如何使用

每个分析类型的 `analysis-primer.md` 都包含：
- 一句话解释（这个分析做什么）
- 能回答什么 / 不能回答什么
- 典型输出文件及其含义
- 常见结果模式和如何解读

根据你需要了解的分析类型，点击下方链接跳转到对应文件。
